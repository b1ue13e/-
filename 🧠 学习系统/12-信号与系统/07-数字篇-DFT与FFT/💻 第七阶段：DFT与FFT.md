## 🧠 为什么需要DFT？

> 理论是完美的，现实是有限的。DFT是计算机能实际计算的变换。

### 从DTFT到DFT

```
DTFT：
X(e^(jΩ)) = Σx[n]e^(-jΩn) （n从-∞到+∞）

问题：
• 无限长序列，计算机无法计算
• Ω连续，计算机无法表示

DFT的解决方案：
• 取有限长：n = 0, 1, ..., N-1
• 频率离散：k = 0, 1, ..., N-1

DFT定义：
X[k] = Σx[n]e^(-j2πkn/N) （n从0到N-1）

记 W_N = e^(-j2π/N) （旋转因子）
X[k] = Σx[n]W_N^(kn)
```

### DFT的隐含周期性

| 主题 | 核心内容 | 状态 | 笔记 |
|:---|:---|:---:|:---|
| **时域周期** | x[n] = x[n+N] | ⬜ | [[DFT时域周期]] |

| **后果** | 循环移位、循环卷积 | ⬜ | [[DFT周期性后果]] |

## 📡 DFT的性质

| 性质 | 时域 | 频域 | 状态 | 笔记 |
|:---|:---|:---|:---:|:---|
| **线性** | ax₁[n] + bx₂[n] | aX₁[k] + bX₂[k] | ⬜ | [[DFT线性]] |
| **循环移位** | x[(n-m) mod N] | X[k]e^(-j2πkm/N) | ⬜ | [[循环移位]] |
| **循环卷积** | x₁[n] ⊛ x₂[n] | X₁[k]·X₂[k] | ⬜ | [[循环卷积]] |
| **乘积** | x₁[n]·x₂[n] | (1/N)X₁[k] ⊛ X₂[k] | ⬜ | [[DFT乘积]] |
| **共轭对称** | x[n]为实数 | X[k] = X*[-k] | ⬜ | [[共轭对称]] |

### 循环卷积 vs 线性卷积

| 类型 | 定义 | 关系 | 状态 | 笔记 |
|:---|:---|:---|:---:|:---|
| **线性卷积** | y[n] = Σx[m]h[n-m] | - | ⬜ | [[线性卷积]] |
| **循环卷积** | y[n] = Σx[m]h[(n-m) mod N] | 周期延拓 | ⬜ | [[循环卷积定义]] |
| **关系** | 循环=线性周期延拓取主周期 | 补零可使相等 | ⬜ | [[卷积关系]] |

```
补零法：
如果x长度L，h长度M，DFT长度N ≥ L+M-1
则循环卷积 = 线性卷积
```

## 🔄 FFT算法

> FFT让DFT从不可能变成瞬间完成。

### 问题：DFT直接计算复杂度O(N²)

```
X[k] = Σx[n]W_N^(kn) （N个k，每个k计算N次乘法）

N=1000时，需要10⁶次运算
N=1000000时，需要10¹²次运算（不可能！）
```

### FFT的解决方案：分治，O(NlogN)

| 主题 | 核心内容 | 状态 | 笔记 |
|:---|:---|:---:|:---|
| **Cooley-Tukey算法** | 基2-FFT，分治策略 | ⬜ | [[Cooley-Tukey]] |
| **分解思想** | N点DFT→2个N/2点DFT | ⬜ | [[FFT分解]] |
| **蝶形运算** | 基本计算单元 | ⬜ | [[蝶形运算]] |
| **位反转** | 输入重排 | ⬜ | [[位反转]] |

```
2点DFT（蝶形运算）：
X[0] = x[0] + x[1]
X[1] = x[0] - x[1]

蝶形图：
    x[0] ──→(+)──→ X[0]
          ↖    ↗
    x[1] ──→(-)──→ X[1]
          W_N^k

8点FFT流程：
输入 → 位反转重排 → 第1级蝶形（2点） → 第2级蝶形（4点） → 第3级蝶形（8点） → 输出

复杂度：
• 级数：log₂N
• 每级蝶形数：N/2
• 总运算量：O(NlogN)

N=1000000时，只需要约20×10⁶次运算！
```

## 🔑 核心公式总结

```
DFT：X[k] = Σx[n]e^(-j2πkn/N) （n从0到N-1）
IDFT：x[n] = (1/N)ΣX[k]e^(j2πkn/N) （k从0到N-1）

复杂度对比：
直接DFT：O(N²)
FFT：O(NlogN)

关键概念：
• DFT隐含周期性
• 循环卷积 vs 线性卷积
• 频谱泄漏与窗函数
• 栅栏效应与补零
```
