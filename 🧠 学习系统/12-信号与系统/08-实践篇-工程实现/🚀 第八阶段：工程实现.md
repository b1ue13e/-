## 🔧 FIR滤波器设计

> FIR特点：有限冲激响应、绝对稳定、可实现线性相位。

### 窗函数法设计步骤

| 步骤 | 内容 | 状态 | 笔记 |
|:---|:---|:---:|:---|
| **1. 理想响应** | 确定H_d(e^(jΩ)) | ⬜ | [[理想响应]] |
| **2. 理想冲激** | h_d[n] = IDTFT{H_d} | ⬜ | [[理想冲激]] |
| **3. 加窗截断** | h[n] = h_d[n]·w[n] | ⬜ | [[加窗]] |
| **4. 因果化** | 移位N/2个样本 | ⬜ | [[因果化]] |

### Python实现

```python
import numpy as np
import matplotlib.pyplot as plt
from scipy import signal

# 设计FIR低通滤波器
N = 101  # 阶数
cutoff = 0.2  # 归一化截止频率

# 使用窗函数法
h = signal.firwin(N, cutoff, window='hamming')

# 计算频率响应
w, H = signal.freqz(h, worN=8000)

# 画图
plt.figure(figsize=(10, 4))
plt.plot(w/np.pi, 20*np.log10(np.abs(H)))
plt.xlabel('Normalized Frequency')
plt.ylabel('Magnitude (dB)')
plt.title('FIR Lowpass Filter Response')
plt.grid(True)
```

## 🔄 IIR滤波器设计

> IIR特点：无限冲激响应、有反馈、阶数低、效率高。

### 常用设计方法

| 方法 | 特点 | 适用场景 | 状态 | 笔记 |
|:---|:---|:---|:---:|:---|
| **巴特沃斯** | 最大平坦，无纹波 | 通带要求平坦 | ⬜ | [[巴特沃斯]] |
| **切比雪夫I型** | 通带等纹波，阻带单调 | 通带要求严格 | ⬜ | [[切比雪夫I]] |
| **切比雪夫II型** | 通带单调，阻带等纹波 | 阻带要求严格 | ⬜ | [[切比雪夫II]] |
| **椭圆滤波器** | 通带阻带都有纹波 | 过渡带最窄 | ⬜ | [[椭圆滤波器]] |

### 双线性变换

| 主题 | 核心内容 | 状态 | 笔记 |
|:---|:---|:---:|:---|
| **变换公式** | s = (2/T)·(1-z^(-1))/(1+z^(-1)) | ⬜ | [[双线性变换]] |
| **优点** | 避免混叠，保证稳定性 | ⬜ | [[双线性优点]] |
| **缺点** | 频率畸变，需预畸变 | ⬜ | [[双线性缺点]] |

### Python实现

```python
from scipy import signal

# 设计IIR巴特沃斯低通滤波器
order = 4
cutoff = 0.2  # 归一化频率

b, a = signal.butter(order, cutoff, btype='low', analog=False)

# 计算频率响应
w, H = signal.freqz(b, a, worN=8000)

# 应用滤波器
y = signal.lfilter(b, a, x)  # x是输入信号
```

```python
import numpy as np
import soundfile as sf
from scipy import signal

# 读取音频
x, fs = sf.read('input.wav')

# 设计带阻滤波器（去除50Hz工频干扰）
f0 = 50  # Hz
Q = 30   # 品质因数
b, a = signal.iirnotch(f0, Q, fs)

# 应用滤波器
y = signal.lfilter(b, a, x)

# 保存结果
sf.write('output.wav', y, fs)
```

### 实战2：频谱分析

```python
import numpy as np
import matplotlib.pyplot as plt
from scipy.fft import fft, fftfreq

# 生成信号
t = np.linspace(0, 1, 1000, endpoint=False)
x = np.sin(2*np.pi*50*t) + 0.5*np.sin(2*np.pi*120*t)

# 加窗
window = np.hamming(len(x))
xw = x * window

# FFT
X = fft(xw)
freqs = fftfreq(len(x), t[1]-t[0])

# 画频谱
plt.plot(freqs[:len(freqs)//2], 20*np.log10(np.abs(X[:len(X)//2])))
plt.xlabel('Frequency (Hz)')
plt.ylabel('Magnitude (dB)')
plt.show()
```
