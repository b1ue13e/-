## 🧠 为什么需要傅里叶变换？

> 时域是表象，频域是本质。傅里叶变换是人类历史上最伟大的"偷懒"。

### 核心思想

```
问题：分析一个复杂信号x(t)

时域方法：直接研究x(t)的波形
• 如果x(t)很复杂，很难看出规律
• 系统响应需要用卷积积分，计算复杂

频域方法：把x(t)拆成正弦波的叠加
• x(t) = ΣX(kω₀)e^(jkω₀t) （傅里叶级数）
• 每个正弦波通过LTI系统后还是正弦波
• 输出 = 各正弦波响应的叠加

为什么这是"偷懒"？
• 时域卷积 → 频域相乘（简单！）
• 微分方程 → 代数方程（简单！）
```

### 棱镜分光类比

```
白光（复杂信号）
    ↓ 通过棱镜（傅里叶变换）
分解成七色光（各频率成分）
    ↓ 通过透镜（系统H(jω)）
各色光被不同程度衰减和相移
    ↓ 再次合成（逆傅里叶变换）
新的光信号（输出y(t)）
```

## ⚙️ 傅里叶变换的性质

| 性质 | 时域 | 频域 | 状态 | 笔记 |
|:---|:---|:---|:---:|:---|
| **线性** | ax₁ + bx₂ | aX₁ + bX₂ | ⬜ | [[线性性质]] |
| **时移** | x(t-t₀) | X(jω)e^(-jωt₀) | ⬜ | [[时移性质]] |
| **频移** | x(t)e^(jω₀t) | X(j(ω-ω₀)) | ⬜ | [[频移性质]] |
| **尺度** | x(at) | (1/\|a\|)X(jω/a) | ⬜ | [[尺度性质]] |
| **卷积定理** | x(t)*h(t) | X(jω)H(jω) | ⬜ | [[卷积定理]] |
| **乘积定理** | x(t)h(t) | (1/2π)X(jω)*H(jω) | ⬜ | [[乘积定理]] |
| **微分** | dx/dt | jωX(jω) | ⬜ | [[微分性质]] |
| **积分** | ∫x(τ)dτ | X(jω)/(jω) + πX(0)δ(ω) | ⬜ | [[积分性质]] |

### 卷积定理的重要性

```
时域：y(t) = x(t) * h(t) = ∫x(τ)h(t-τ)dτ （难算！）

频域：Y(jω) = X(jω) · H(jω) （简单！）

系统频率响应H(jω)：
• |H(jω)|：幅度响应（各频率通过的比例）
• arg(H(jω))：相位响应（各频率的相移）
```

## 🔑 核心公式总结

```
傅里叶变换：X(jω) = ∫x(t)e^(-jωt)dt
逆变换：x(t) = (1/2π)∫X(jω)e^(jωt)dω

指数信号：e^(-at)u(t) ↔ 1/(jω + a)
矩形↔sinc：rect(t/τ) ↔ τsinc(ωτ/2)

卷积定理：x(t)*h(t) ↔ X(jω)H(jω)
```
