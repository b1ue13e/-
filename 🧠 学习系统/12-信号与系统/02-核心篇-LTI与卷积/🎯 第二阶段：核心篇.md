## 🧠 卷积的本质

> 卷积是信号与系统的灵魂。不理解卷积，后面所有变换都是空中楼阁。

### 故事版解释

```
想象你在敲击一个鼓：

第1秒敲一下 → 鼓发出"咚咚"声（持续2秒衰减）
第2秒敲一下 → 鼓又发出"咚咚"声
第3秒敲一下 → 鼓再发出"咚咚"声

你听到的总声音 = 三次敲击的响应叠加

每次敲击的强度不同（权重）
每次响应有时间延迟（时移）

这就是卷积：加权 + 叠加
```

### 数学定义

| 类型 | 公式 | 状态 | 笔记 |
|:---|:---|:---:|:---|
| **连续卷积** | y(t) = ∫x(τ)h(t-τ)dτ | ⬜ | [[连续卷积]] |
| **离散卷积** | y[n] = Σx[k]h[n-k] | ⬜ | [[离散卷积]] |
| **变量说明** | τ/k：历史时刻；t-τ/n-k：时间差 | ⬜ | [[卷积变量]] |

## 🔄 卷积的计算方法

### 图解法

| 步骤 | 操作 | 状态 | 笔记 |
|:---|:---|:---:|:---|
| **换元** | x(t)→x(τ), h(t)→h(τ) | ⬜ | [[图解法换元]] |
| **反转** | h(τ)→h(-τ) | ⬜ | [[图解法反转]] |
| **平移** | h(-τ)→h(t-τ) | ⬜ | [[图解法平移]] |
| **相乘** | x(τ)·h(t-τ) | ⬜ | [[图解法相乘]] |
| **积分** | 对乘积曲线下面积积分 | ⬜ | [[图解法积分]] |
| **滑动** | t从-∞到+∞重复 | ⬜ | [[图解法滑动]] |

### 解析法

| 性质 | 公式 | 物理意义 | 状态 | 笔记 |
|:---|:---|:---|:---:|:---|
| **与δ卷积** | x(t)*δ(t) = x(t) | 恒等运算 | ⬜ | [[卷积恒等]] |
| **与δ(t-t₀)卷积** | x(t)*δ(t-t₀) = x(t-t₀) | 时移 | ⬜ | [[卷积时移]] |
| **与u(t)卷积** | x(t)*u(t) = ∫x(τ)dτ | 积分器 | ⬜ | [[卷积积分]] |

## 🔑 核心公式总结

```
卷积定义：y(t) = ∫x(τ)h(t-τ)dτ

与冲激：x(t)*δ(t-t₀) = x(t-t₀)
与阶跃：x(t)*u(t) = ∫x(τ)dτ （积分）

因果系统：h(t) = 0, t < 0
稳定系统：∫|h(t)|dt < ∞
```
