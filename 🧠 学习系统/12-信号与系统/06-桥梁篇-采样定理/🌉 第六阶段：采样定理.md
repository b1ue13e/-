## 🧠 采样的数学模型

> 采样定理是连接模拟世界和数字世界的桥梁。

### 采样过程

```
连续信号 x(t)
    ↓ 采样（间隔Tₛ，频率fₛ=1/Tₛ）
离散信号 x[n] = x(nTₛ)

数学表示：
xₛ(t) = x(t) · p(t)

其中p(t)是冲激串：
p(t) = Σδ(t-nTₛ) （n从-∞到+∞）
```

### 频域效应（关键！）

```
时域相乘 ↔ 频域卷积

Xₛ(jω) = (1/2π) X(jω) * P(jω)

冲激串的傅里叶变换：
P(jω) = (2π/Tₛ) Σδ(ω - nωₛ) ，ωₛ=2π/Tₛ

所以：
Xₛ(jω) = (1/Tₛ) ΣX(j(ω - nωₛ))

关键结论：
采样使频谱以ωₛ为周期重复！
```

### 混叠（Aliasing）

| 主题 | 核心内容 | 状态 | 笔记 |
|:---|:---|:---:|:---|
| **产生条件** | fₛ < 2fₘ时频谱重叠 | ⬜ | [[混叠产生]] |
| **后果** | 高频"伪装"成低频 | ⬜ | [[混叠后果]] |
| **避免方法** | 抗混叠滤波器 | ⬜ | [[抗混叠]] |

```
当 fₛ < 2fₘ 时，频谱重叠：

    ▲
    │    ┌───┐┌───┐
    │   /     XX     \
    │  /       XX      \
    └─┬───┬───┬───┬──→ ω
     -ωₛ   0   ωₛ
           ↑
        重叠区（混叠）

后果：
• 高频成分"伪装"成低频成分
• 无法区分原始频率
• 信号失真，无法完美重建
```

### 重建过程

```
理想重建步骤：
1. 让采样信号通过理想低通滤波器
2. 截止频率：ωₘ < ωc < ωₛ - ωₘ
3. 增益：Tₛ

效果：
滤除所有高频重复分量
只保留基带频谱

时域解释：sinc插值
x(t) = Σx[n]sinc((t-nTₛ)/Tₛ)

每个采样点用一个sinc函数表示
所有sinc叠加就是重建信号
```

## ⚙️ 抗混叠滤波

```
实际信号往往有高频噪声，怎么办？

模拟信号 x(t)（含高频噪声）
    ↓ 抗混叠滤波器（模拟低通）
滤波后信号 xₐ(t)（最高频率fₘ）
    ↓ 采样（fₛ > 2fₘ）
采样信号 x[n]
    ↓ 数字处理
...

关键：
必须在采样前滤除高于fₛ/2的频率！
否则混叠后无法区分
```
