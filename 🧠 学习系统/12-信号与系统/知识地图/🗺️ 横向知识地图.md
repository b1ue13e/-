# 🗺️ 信号与系统横向知识地图

> 从四个维度构建完整的信号与系统知识框架

---

## 1. ⏱️ 时域维度

### 1.1 信号表示
| 概念 | 描述 | 关键公式 |
|:---|:---|:---|
| **连续时间信号** | 定义在连续时间域上的信号 | $x(t), t \in \mathbb{R}$ |
| **离散时间信号** | 定义在离散时间点上的序列 | $x[n], n \in \mathbb{Z}$ |
| **基本信号** | 冲激、阶跃、指数、正弦 | $\delta(t), u(t), e^{st}, \cos(\omega t)$ |
| **信号运算** | 时移、尺度变换、反褶 | $x(t-t_0), x(at), x(-t)$ |
| **信号分解** | 奇偶分解、冲激分解 | $x(t) = x_e(t) + x_o(t)$ |

### 1.2 系统响应
| 概念 | 描述 | 关键公式 |
|:---|:---|:---|
| **零输入响应** | 仅由初始状态引起的响应 | $y_{zi}(t)$ |
| **零状态响应** | 仅由输入信号引起的响应 | $y_{zs}(t)$ |
| **完全响应** | 零输入+零状态响应 | $y(t) = y_{zi}(t) + y_{zs}(t)$ |
| **冲激响应** | 系统对冲激信号的响应 | $h(t)$ 或 $h[n]$ |
| **阶跃响应** | 系统对阶跃信号的响应 | $s(t)$ 或 $s[n]$ |

### 1.3 卷积运算
```
连续时间卷积:                    离散时间卷积:
┌─────────────────┐              ┌─────────────────┐
│  ∞              │              │  ∞              │
│  ∫ x(τ)h(t-τ)dτ │              │  Σ x[k]h[n-k]   │
│ -∞              │              │ k=-∞            │
└─────────────────┘              └─────────────────┘
```

**卷积性质**: 交换律、结合律、分配律、时移性质

### 1.4 差分方程
| 类型 | 形式 | 求解方法 |
|:---|:---|:---|
| **线性常系数差分方程** | $\sum_{k=0}^{N} a_k y[n-k] = \sum_{k=0}^{M} b_k x[n-k]$ | 齐次解+特解 |
| **递归系统** | 输出依赖于过去输出 | IIR滤波器 |
| **非递归系统** | 输出仅依赖于输入 | FIR滤波器 |

## 2. 📊 频域维度

### 2.1 傅里叶分析
| 变换类型 | 适用信号 | 变换对 |
|:---|:---|:---|
| **连续傅里叶级数 (CFS)** | 周期连续信号 | $x(t) = \sum_{k=-\infty}^{\infty} c_k e^{jk\omega_0 t}$ |
| **连续傅里叶变换 (CTFT)** | 非周期连续信号 | $X(j\omega) = \int_{-\infty}^{\infty} x(t)e^{-j\omega t}dt$ |
| **离散傅里叶级数 (DFS)** | 周期离散序列 | $x[n] = \sum_{k=0}^{N-1} c_k e^{j\frac{2\pi}{N}kn}$ |
| **离散时间傅里叶变换 (DTFT)** | 非周期离散序列 | $X(e^{j\omega}) = \sum_{n=-\infty}^{\infty} x[n]e^{-j\omega n}$ |
| **离散傅里叶变换 (DFT)** | 有限长序列 | $X[k] = \sum_{n=0}^{N-1} x[n]e^{-j\frac{2\pi}{N}kn}$ |

### 2.2 频谱特性
```
频谱组成:
┌─────────────────────────────────────────┐
│  X(f) = |X(f)| · e^{j∠X(f)}            │
│         ↑           ↑                   │
│     幅度谱      相位谱                   │
│  (Amplitude)   (Phase)                  │
└─────────────────────────────────────────┘
```

| 特性 | 说明 |
|:---|:---|
| **幅度谱** | 信号各频率分量的强度 |
| **相位谱** | 信号各频率分量的相位关系 |
| **功率谱密度** | 单位频率内的功率分布 |
| **带宽** | 信号占据的有效频率范围 |

### 2.3 滤波概念
| 滤波器类型 | 特性 | 理想频率响应 |
|:---|:---|:---|
| **低通滤波器 (LPF)** | 通过低频，衰减高频 | $H(j\omega) = \begin{cases} 1 & \lvert \omega \rvert \leq \omega_c \\ 0 & \lvert \omega \rvert > \omega_c \end{cases}$ |
| **高通滤波器 (HPF)** | 通过高频，衰减低频 | $H(j\omega) = \begin{cases} 0 & \lvert \omega \rvert \leq \omega_c \\ 1 & \lvert \omega \rvert > \omega_c \end{cases}$ |
| **带通滤波器 (BPF)** | 通过特定频带 | $H(j\omega) = \begin{cases} 1 & \omega_1 \leq \lvert \omega \rvert \leq \omega_2 \\ 0 & \text{其他} \end{cases}$ |
| **带阻滤波器 (BSF)** | 衰减特定频带 | 带通的互补 |

### 2.4 频率响应
```
系统频率响应:
┌─────────────────────────────────────────┐
│                                         │
│   H(jω) = Y(jω) / X(jω)                 │
│         = |H(jω)| · e^{j∠H(jω)}         │
│                                         │
│   幅频特性: |H(jω)| vs ω                 │
│   相频特性: ∠H(jω) vs ω                  │
│                                         │
└─────────────────────────────────────────┘
```

## 3. 🔄 变换维度

### 3.1 拉普拉斯变换
| 项目 | 内容 |
|:---|:---|
| **定义** | $X(s) = \int_{-\infty}^{\infty} x(t)e^{-st}dt$, 其中 $s = \sigma + j\omega$ |
| **收敛域 (ROC)** | 使积分收敛的 $s$ 的取值范围 |
| **单边变换** | $X(s) = \int_{0^{-}}^{\infty} x(t)e^{-st}dt$（因果系统） |
| **逆变换** | $x(t) = \frac{1}{2\pi j}\int_{\sigma-j\infty}^{\sigma+j\infty} X(s)e^{st}ds$ |

**常用变换对**:
| 信号 $x(t)$ | 变换 $X(s)$ | ROC |
|:---|:---|:---|
| $\delta(t)$ | $1$ | 全 $s$ 平面 |
| $u(t)$ | $\frac{1}{s}$ | $\text{Re}\{s\} > 0$ |
| $e^{-at}u(t)$ | $\frac{1}{s+a}$ | $\text{Re}\{s\} > -a$ |
| $t^n u(t)$ | $\frac{n!}{s^{n+1}}$ | $\text{Re}\{s\} > 0$ |
| $\cos(\omega_0 t)u(t)$ | $\frac{s}{s^2+\omega_0^2}$ | $\text{Re}\{s\} > 0$ |

### 3.2 Z变换
| 项目 | 内容 |
|:---|:---|
| **定义** | $X(z) = \sum_{n=-\infty}^{\infty} x[n]z^{-n}$ |
| **收敛域 (ROC)** | 使级数收敛的 $z$ 的取值范围（环形区域） |
| **单边Z变换** | $X(z) = \sum_{n=0}^{\infty} x[n]z^{-n}$（因果序列） |
| **逆变换** | 围线积分、部分分式展开、长除法 |

**常用变换对**:
| 序列 $x[n]$ | 变换 $X(z)$ | ROC |
|:---|:---|:---|
| $\delta[n]$ | $1$ | 全 $z$ 平面 |
| $u[n]$ | $\frac{1}{1-z^{-1}}$ | $\lvert z \rvert > 1$ |
| $a^n u[n]$ | $\frac{1}{1-az^{-1}}$ | $\lvert z \rvert > \lvert a \rvert$ |
| $na^n u[n]$ | $\frac{az^{-1}}{(1-az^{-1})^2}$ | $\lvert z \rvert > \lvert a \rvert$ |
| $\cos(\omega_0 n)u[n]$ | $\frac{1-\cos(\omega_0)z^{-1}}{1-2\cos(\omega_0)z^{-1}+z^{-2}}$ | $\lvert z \rvert > 1$ |

### 3.3 收敛域 (ROC) 性质
```
拉普拉斯变换 ROC 类型:          Z变换 ROC 类型:
┌──────────────────┐           ┌──────────────────┐
│  右半平面         │           │  圆外区域         │
│  Re{s} > σ₀      │           │  |z| > R₁        │
│  ←因果信号        │           │  ←右边序列        │
├──────────────────┤           ├──────────────────┤
│  左半平面         │           │  圆内区域         │
│  Re{s} < σ₀      │           │  |z| < R₂        │
│  ←反因果信号      │           │  ←左边序列        │
├──────────────────┤           ├──────────────────┤
│  带状区域         │           │  环形区域         │
│  σ₁ < Re{s} < σ₂ │           │  R₁ < |z| < R₂   │
│  ←双边信号        │           │  ←双边序列        │
└──────────────────┘           └──────────────────┘
```

### 3.4 系统函数
| 变换域 | 系统函数 | 与冲激响应关系 |
|:---|:---|:---|
| **拉普拉斯域** | $H(s) = \frac{Y(s)}{X(s)} = \frac{\sum_{k=0}^{M} b_k s^k}{\sum_{k=0}^{N} a_k s^k}$ | $H(s) = \mathcal{L}\{h(t)\}$ |
| **Z域** | $H(z) = \frac{Y(z)}{X(z)} = \frac{\sum_{k=0}^{M} b_k z^{-k}}{\sum_{k=0}^{N} a_k z^{-k}}$ | $H(z) = \mathcal{Z}\{h[n]\}$ |

**系统函数与稳定性**:
- 连续系统稳定 ⇔ $H(s)$ 的所有极点位于左半平面
- 离散系统稳定 ⇔ $H(z)$ 的所有极点位于单位圆内

## 4. 💻 应用维度

### 4.1 采样理论
| 概念 | 内容 | 关键公式 |
|:---|:---|:---|
| **理想采样** | 用冲激串对连续信号采样 | $x_s(t) = x(t) \cdot \sum_{n=-\infty}^{\infty} \delta(t-nT_s)$ |
| **采样定理** | 奈奎斯特采样定理 | $f_s \geq 2f_{max}$ |
| **混叠现象** | 采样频率不足导致频谱重叠 | 抗混叠滤波器 |
| **信号重建** | 从采样值恢复原信号 | 理想低通滤波（sinc插值） |
| **量化** | 幅度离散化 | $x_Q[n] = Q\{x[n]\}$ |

```
采样与重建过程:
┌─────────┐    采样     ┌─────────┐    量化     ┌─────────┐
│ x(t)    │ ─────────→ │ x[n]    │ ─────────→ │ x_Q[n]  │
│ (连续)   │            │ (离散)   │            │ (数字)   │
└─────────┘            └─────────┘            └─────────┘
                                              ↓
┌─────────┐    重建     ┌─────────┐    D/A    ┌─────────┐
│ x̂(t)    │ ←───────── │ 插值    │ ←──────── │ 解码    │
│ (连续)   │            │         │            │         │
└─────────┘            └─────────┘            └─────────┘
```

### 4.2 数字滤波
| 类型 | 特点 | 设计方法 |
|:---|:---|:---|
| **FIR滤波器** | 有限冲激响应、线性相位、稳定 | 窗函数法、频率采样法、最优化设计 |
| **IIR滤波器** | 无限冲激响应、非线性相位、高效 | 模拟原型法（巴特沃斯、切比雪夫、椭圆） |
| **自适应滤波** | 系数自动调整 | LMS算法、RLS算法 |
| **多速率处理** | 抽取、插值、采样率转换 | 抗镜像滤波、抗混叠滤波 |

**FIR vs IIR 对比**:
| 特性 | FIR | IIR |
|:---|:---|:---|
| 稳定性 | 总是稳定 | 需要验证极点位置 |
| 相位特性 | 可实现严格线性相位 | 非线性相位 |
| 实现复杂度 | 需要较高阶数 | 较低阶数即可达到同样效果 |
| 设计难度 | 相对简单 | 较为复杂 |

### 4.3 信号处理应用
| 应用领域 | 典型应用 | 涉及技术 |
|:---|:---|:---|
| **通信系统** | 调制解调、信道均衡、扩频 | 带通采样、匹配滤波、OFDM |
| **音频处理** | 音频编码、降噪、均衡器 | 子带编码、谱减法、均衡 |
| **图像处理** | 图像滤波、边缘检测、压缩 | 2D-DFT、小波变换、JPEG |
| **生物医学** | ECG/EEG分析、医学成像 | 时频分析、滤波、重建 |
| **雷达系统** | 脉冲压缩、目标检测 | 匹配滤波、FFT处理 |
| **控制系统** | 系统辨识、状态估计 | 卡尔曼滤波、频域分析 |

### 4.4 工程实现
| 实现方式 | 平台/工具 | 特点 |
|:---|:---|:---|
| **DSP芯片** | TI TMS320系列、ADSP系列 | 专用硬件、实时处理 |
| **FPGA实现** | Xilinx、Altera | 并行处理、低延迟 |
| **软件实现** | MATLAB、Python (SciPy) | 快速原型、算法验证 |
| **嵌入式系统** | ARM Cortex-M/A | 低功耗、便携式 |

**实现考虑因素**:
- 有限字长效应（量化噪声、溢出）
- 实时性要求
- 功耗与成本
- 精度与速度权衡

## 🔄 维度间联系

```
                    ┌─────────────┐
                    │   ⏱️ 时域    │
                    │  (t 或 n)   │
                    └──────┬──────┘
                           │ 傅里叶变换
                           ↓
┌─────────────┐      ┌─────────────┐      ┌─────────────┐
│  🔄 变换域   │ ←──→ │   📊 频域   │ ←──→ │  💻 应用    │
│ (s 或 z域)   │      │  (ω 或 Ω)   │      │   实现      │
└─────────────┘      └─────────────┘      └─────────────┘
       ↑                                          │
       └────────── 系统分析与设计 ←───────────────┘
```

### 维度转换关系总结
| 从 \ 到 | 时域 | 频域 | 变换域 |
|:---|:---|:---|:---|
| **时域** | - | 傅里叶变换 | 拉普拉斯/Z变换 |
| **频域** | 逆傅里叶变换 | - | 代入 $s=j\omega$ 或 $z=e^{j\omega}$ |
| **变换域** | 逆变换 | 沿虚轴/单位圆求值 | - |
