## 🧠 为什么需要拉普拉斯变换？

> 傅里叶变换是理想世界的王子，拉普拉斯变换是现实世界的骑士。

### 傅里叶变换的局限

```
傅里叶变换要求：∫|x(t)|dt < ∞ （绝对可积）

但许多重要信号不满足：
• x(t) = e^(2t)u(t) （增长指数，积分发散）
• x(t) = u(t) （阶跃，积分发散）
• x(t) = cos(t)u(t) （无始无终，积分发散）

解决方案：给信号乘一个衰减因子 e^(-σt)！
```

## 📡 收敛域（ROC）

> ROC是拉普拉斯的灵魂，没有ROC，变换不完整！

| 主题 | 核心内容 | 状态 | 笔记 |
|:---|:---|:---:|:---|
| **ROC定义** | 使积分收敛的所有s值的集合 | ⬜ | [[ROC定义]] |
| **右边信号** | ROC: Re(s) > σ₀（极点右侧） | ⬜ | [[右边信号ROC]] |
| **左边信号** | ROC: Re(s) < σ₀（极点左侧） | ⬜ | [[左边信号ROC]] |
| **双边信号** | ROC: σ₁ < Re(s) < σ₂（带状） | ⬜ | [[双边信号ROC]] |
| **有限时长** | ROC: 全s平面 | ⬜ | [[有限时长ROC]] |

### 重要结论

```
相同的X(s)，不同的ROC，对应不同的时域信号！

例：X(s) = 1/(s+a)
• ROC: Re(s) > -a → x(t) = e^(-at)u(t) （因果）
• ROC: Re(s) < -a → x(t) = -e^(-at)u(-t) （非因果）
```

### 稳定性判据

```
因果LTI系统稳定的充要条件：
所有极点都在s左半平面（实部 < 0）

为什么？
h(t) = ΣAᵢe^(pᵢt)u(t)
如果Re(pᵢ) < 0，指数衰减，∫|h(t)|dt < ∞
如果Re(pᵢ) > 0，指数增长，系统不稳定
```

## ⚙️ 与傅里叶变换的关系

| 主题 | 核心内容 | 状态 | 笔记 |
|:---|:---|:---:|:---|
| **特殊关系** | X(jω) = X(s)\|_{s=jω} | ⬜ | [[s=jw]] |
| **存在条件** | jω必须在ROC内 | ⬜ | [[存在条件]] |
| **例子分析** | e^(-at)u(t) vs e^(at)u(t) | ⬜ | [[对比分析]] |

```
傅里叶变换是拉普拉斯变换的特例！

例：e^(-at)u(t)，a>0
• X(s) = 1/(s+a)，ROC: Re(s) > -a
• jω在ROC内（因为Re(jω)=0>-a）
• 所以傅里叶变换存在：X(jω) = 1/(jω+a)

反例：e^(at)u(t)，a>0
• X(s) = 1/(s-a)，ROC: Re(s) > a
• jω不在ROC内（Re(jω)=0 < a）
• 傅里叶变换不存在！
```

- [ ] 理解s=jω的几何意义
