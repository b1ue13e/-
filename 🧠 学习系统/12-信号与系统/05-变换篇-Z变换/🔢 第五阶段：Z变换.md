## 🧠 Z变换是什么？

> 如果说拉普拉斯是连续系统的瑞士军刀，Z变换就是离散系统的瑞士军刀。

### 从拉普拉斯到Z变换

```
连续系统：拉普拉斯变换，s域
离散系统：Z变换，z域

联系：z = e^(sT) （T是采样周期）

推导：
对采样信号x[n] = x(nT)做拉普拉斯变换：
X(s) = Σx[n]e^(-snT)

令 z = e^(sT)，则：
X(z) = Σx[n]z^(-n)

这就是Z变换！
```

### Z平面的含义

| 属性 | 含义 | 对应s平面 | 状态 | 笔记 |
|:---|:---|:---|:---:|:---|
| **z = re^(jΩ)** | r：半径，Ω：数字频率 | - | ⬜ | [[Z平面]] |
| **单位圆内(r<1)** | 衰减信号 | s左半平面 | ⬜ | [[单位圆内]] |
| **单位圆外(r>1)** | 增长信号 | s右半平面 | ⬜ | [[单位圆外]] |
| **单位圆上(r=1)** | 等幅振荡 | s虚轴 | ⬜ | [[单位圆上]] |

## 📡 Z变换的定义与ROC

| 主题 | 核心内容 | 状态 | 笔记 |
|:---|:---|:---:|:---|
| **双边Z变换** | X(z) = Σx[n]z^(-n) | ⬜ | [[双边Z变换]] |
| **单边Z变换** | X(z) = Σx[n]z^(-n) (n≥0) | ⬜ | [[单边Z变换]] |
| **ROC定义** | 使级数收敛的所有z值的集合 | ⬜ | [[Z变换ROC]] |

### 常见序列的Z变换

| 序列 x[n] | Z变换 X(z) | ROC | 状态 | 笔记 |
|:---|:---|:---|:---:|:---|
| δ[n] | 1 | 全z平面 | ⬜ | [[冲激Z变换]] |
| u[n] | z/(z-1) | \|z\| > 1 | ⬜ | [[阶跃Z变换]] |
| aⁿu[n] | z/(z-a) | \|z\| > \|a\| | ⬜ | [[指数Z变换]] |
| -aⁿu[-n-1] | z/(z-a) | \|z\| < \|a\| | ⬜ | [[左边Z变换]] |

**注意**：相同的X(z)，不同的ROC，对应不同的序列！

### 稳定性判据

```
因果离散LTI系统稳定的充要条件：
所有极点都在单位圆内（|pᵢ| < 1）

为什么？
h[n] = ΣAᵢ(pᵢ)ⁿu[n]
如果|pᵢ| < 1，指数衰减，Σ|h[n]| < ∞
如果|pᵢ| > 1，指数增长，系统不稳定

对比连续系统：
连续：极点在s左半平面
离散：极点在z单位圆内
```

## ⚙️ 与DTFT的关系

| 主题 | 核心内容 | 状态 | 笔记 |
|:---|:---|:---:|:---|
| **DTFT定义** | X(e^(jΩ)) = Σx[n]e^(-jΩn) | ⬜ | [[DTFT定义]] |
| **与Z关系** | X(e^(jΩ)) = X(z)\|_{z=e^(jΩ)} | ⬜ | [[Z与DTFT]] |
| **存在条件** | 单位圆必须在ROC内 | ⬜ | [[DTFT存在条件]] |
| **周期性** | H(e^(j(Ω+2π))) = H(e^(jΩ)) | ⬜ | [[DTFT周期性]] |

```
关键理解：
• DTFT是Z变换在单位圆上的取值
• 离散系统频率响应以2π为周期
• 数字频率Ω与模拟频率ω的关系：Ω = ωT
```

## 🔑 核心公式总结

```
Z变换：X(z) = Σx[n]z^(-n)

常见变换对：
u[n] ↔ z/(z-1), ROC: |z| > 1
aⁿu[n] ↔ z/(z-a), ROC: |z| > |a|

稳定性：因果离散系统稳定 ⇔ 所有极点在单位圆内

与DTFT关系：X(e^(jΩ)) = X(z)|_{z=e^(jΩ)}
```
