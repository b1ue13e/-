## 🎯 诊断目的

信号与系统是一门**极度依赖前置知识**的课程。如果基础不牢，会在以下地方崩盘：

| 知识点 | 需要的前置技能 | 崩溃点 |
|:---|:---|:---|
| 卷积积分 | 微积分（换元、分部积分） | 算不出积分 |
| 傅里叶级数 | 复数欧拉公式、正交性 | 看不懂频谱 |
| 系统函数 | 复变函数、留数定理 | 不会逆变换 |
| 状态空间 | 线性代数（矩阵指数） | 解不了方程 |

**本诊断将帮助你**：
1. 确认当前水平（同步学习 / 回炉重造 / 专项突破）
2. 找出薄弱环节（需要补课的地方）
3. 制定学习策略（从哪里开始，pace 如何）

- [ ] **D. 兴趣自学** - 纯粹为了理解技术底层

**Q2：你目前学到哪个阶段？**
- [ ] **完全零基础** - 还没开始，或第一节课就听不懂
- [ ] **时域挣扎** - 信号、系统、卷积的概念混乱
- [ ] **频域迷茫** - 傅里叶变换的公式会背，但不知道在干什么
- [ ] **变换混乱** - 傅里叶/拉普拉斯/Z变换混在一起，分不清
- [ ] **采样困惑** - 采样定理理解不了，混叠概念模糊
- [ ] **数字卡壳** - DFT/FFT的计算和物理意义对不上
- [ ] **整体混乱** - 好像都学过，但串不起来

**Q3：你的目标是什么？**
- [ ] **应付考试** - 期末/考研及格就行
- [ ] **理解原理** - 想真正搞懂，用于后续课程（通信、控制、DSP）
- [ ] **工程应用** - 工作需要，要做滤波器、信号处理
- [ ] **学术研究** - 打算深入研究某个方向

### Part 2：数学装备检查

**Section A：微积分（必须过关）**

**Q4：以下积分，你能算出来吗？**

```
∫ e^(-t) * sin(t) dt  （从0到∞）
```

- [ ] **A. 轻松搞定** - 知道用分部积分或拉普拉斯
- [ ] **B. 有点吃力** - 能做出来，但要花点时间
- [ ] **C. 完全不会** - 不知道从何下手

**Q5：微分方程的解的结构，你理解吗？**

```
y'' + 3y' + 2y = x(t)
```

- [ ] **A. 完全理解** - 齐次解+特解，特征方程，都会
- [ ] **B. 有点模糊** - 好像记得齐次解和特解，但分不清楚
- [ ] **C. 完全不懂** - 这是啥？

**Section B：线性代数（必须过关）**

**Q6：矩阵乘法、行列式、逆矩阵，你熟练吗？**

- [ ] **A. 非常熟练** - 闭眼都能算
- [ ] **B. 基本会算** - 慢一点，但能算对
- [ ] **C. 基本忘了** - 需要翻书回忆

**Q7：特征值和特征向量的物理意义？**

- [ ] **A. 完全理解** - 知道是"不改变方向只改变长度"的变换
- [ ] **B. 有点模糊** - 会算，但不知道有什么用
- [ ] **C. 完全不懂** - 这是啥？

**Section C：复数（必须过关）**

**Q8：欧拉公式 $e^{j\theta} = \cos\theta + j\sin\theta$，你理解吗？**

```
问题：e^(jπ) = ?
```

- [ ] **A. 秒答** - 等于-1，欧拉恒等式
- [ ] **B. 能推导** - 代入θ=π，得到cosπ + jsinπ = -1
- [ ] **C. 看不懂** - j是什么？为什么指数里能有虚数？

**Q9：复数的极坐标表示，你能熟练转换吗？**

```
1 + j = ? (极坐标形式)
```

- [ ] **A. 轻松转换** - √2 * e^(jπ/4)
- [ ] **B. 有点吃力** - 知道公式，但算得慢
- [ ] **C. 完全不会** - 什么极坐标？

## 🔍 诊断结果分析

### 评分标准

**数学基础分**（Q4-Q9，每题A=2分，B=1分，C=0分）：
- **12分**：数学装备精良，可以直接开干
- **8-11分**：数学有基础，但个别地方需要补
- **4-7分**：数学薄弱，建议先补课
- **0-3分**：数学零基础，必须补课

**概念理解分**（Q10-Q11，每题A=2分，B=1分，C=0分）：
- **4分**：概念清晰，只需要系统梳理
- **2-3分**：概念模糊，需要加强物理直觉
- **0-1分**：概念空白，从头开始

### 典型画像

| 类型 | 特征 | 建议路线 |
|:---|:---|:---|
| **🔥 满血战士** | 数学12分+概念4分 | 从第3阶段开始，快速过基础，重点突破频域 |
| **⚔️ 装备不全** | 数学8-11分+概念2-3分 | 正常推进，数学薄弱环节同步补 |
| **🛡️ 新兵蛋子** | 数学4-7分+概念0-2分 | 必须先从第1阶段开始，数学先补课 |
| **💀 濒死状态** | 数学0-3分+概念0-1分 | 先学微积分和复数，再谈信号系统 |

## 📚 补课建议（按需服用）

### 如果微积分薄弱

**必补内容**：
1. **积分技巧**：换元积分、分部积分
2. **广义积分**：无穷限积分、含参变量积分
3. **微分方程**：一阶/二阶线性常系数微分方程

**推荐资源**：
- [[🧠 学习系统/06-常微分方程/🗺️ 常微分方程学习路径|常微分方程学习路径]]
- 任何一本高数教材的"微分方程"章节

### 如果复数薄弱

**必补内容**：
1. **复数四则运算**
2. **欧拉公式**：$e^{j\theta} = \cos\theta + j\sin\theta$
3. **极坐标与直角坐标转换**

**快速补课**（2小时）：
- 画单位圆，标出 $e^{j\theta}$ 的几何意义
- 推导欧拉公式（用泰勒展开）
- 练习：把几个复数在极坐标和直角坐标间转换

### 如果线性代数薄弱

**必补内容**：
1. **矩阵运算**：乘法、转置、逆
2. **特征值/特征向量**：定义、计算、物理意义

**推荐资源**：
- [[🧠 学习系统/09-高等代数/01-基础入门/📐 第一阶段：基础入门|高等代数 第一阶段]]（矩阵部分）
