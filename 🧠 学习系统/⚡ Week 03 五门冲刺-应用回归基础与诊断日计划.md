# ⚡ Week 03 五门冲刺：应用回归基础与诊断日计划

> 适用日期：**2026-05-02 ~ 2026-05-08**
>
> 对应周次：[[✅ 2026-04-18 ~ 2026-07-03 五门冲刺每周清单|五门冲刺 Week 03]]
>
> 本周主课：[[30-应用回归分析/🗺️ 应用回归分析学习路径|应用回归分析]]

---

## 这周真正要完成什么

这周不是“把所有回归模型学完”，而是把应用回归最核心的一条主线真正打通：

`建模问题 -> 一元/多元线性回归 -> 最小二乘 -> 显著性检验 -> 残差分析 -> 诊断与修正 -> 变量选择与共线性`

本周结束时，你至少要能做到：

1. 说清回归是在解决什么问题，不再把它理解成“拟合一条线”。
2. 理解一元和多元线性回归的共同骨架。
3. 说清为什么回归必须同时看“估计、检验、残差、预测”。
4. 理解异方差、自相关、异常点、变量选择、多重共线性分别在提醒什么风险。

本周不要求：

- 推完所有统计证明
- 把每个诊断方法都做到程序级实现
- 深入因果和第四阶段外延专题

---

## 本周主线

建议你始终围绕这一条线学：

`回归分析概述 -> 最小二乘 -> 一元回归 -> 多元回归 -> 显著性检验 -> 残差分析 -> 异方差/自相关/异常值 -> 变量选择 -> 多重共线性`

---

## 日执行规则

- 本周主线是“线性回归 + 诊断”，不要提前跳进第三阶段扩展回归。
- 如果某天状态一般，优先保“模型理解 + 1 个输出”，不追求笔记数量。
- 这周最重要的不是新术语数量，而是你能不能讲清回归建模流程。
