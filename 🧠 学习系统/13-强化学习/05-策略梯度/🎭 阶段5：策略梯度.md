## 📐 策略梯度定理

> 策略优化的理论基础。

| 主题 | 核心内容 | 状态 | 笔记 |
|:---|:---|:---:|:---|
| **目标函数** | J(θ) = E[ Σ log π_θ(a_t\|s_t) * G_t ] | ⬜ | [[PG-目标函数]] |
| **策略梯度** | ∇_θ J(θ) = E[ Σ ∇_θ log π_θ(a_t\|s_t) * G_t ] | ⬜ | [[PG-梯度]] |
| **似然比技巧** | ∇p(x) = p(x)∇log p(x) | ⬜ | [[PG-似然比]] |
| **直观解释** | 高回报动作增加概率，低回报减少概率 | ⬜ | [[PG-直观]] |

### 梯度推导关键步骤

```
目标: ∇_θ E[G_t]

= E[ G_t * ∇_θ log π_θ(a_t|s_t) ]

解释:
- G_t > 0: 增加该动作概率
- G_t < 0: 减少该动作概率
- |G_t|大: 更新幅度大
```

## 🔄 REINFORCE算法

> 蒙特卡洛策略梯度算法。

| 主题 | 核心内容 | 状态 | 笔记 |
|:---|:---|:---:|:---|
| **蒙特卡洛采样** | 使用当前策略采样完整轨迹 | ⬜ | [[REINFORCE-MC]] |
| **回报计算** | G_t = Σ_{k=t}^T γ^{k-t} r_k | ⬜ | [[REINFORCE-回报]] |
| **梯度更新** | θ ← θ + α * ∇_θ log π_θ(a_t\|s_t) * G_t | ⬜ | [[REINFORCE-更新]] |
| **无偏估计** | 梯度的期望等于真实梯度 | ⬜ | [[REINFORCE-无偏]] |

### REINFORCE算法流程

```python
for episode:
    采样轨迹 τ = (s_0, a_0, r_1, ..., s_T)
    for t in range(T):
        G_t = Σ_{k=t}^T γ^{k-t} r_k  # 计算回报
    
    for t in range(T):
        loss += -log π_θ(a_t|s_t) * G_t
    
    θ ← θ - α * ∇_θ loss
```
