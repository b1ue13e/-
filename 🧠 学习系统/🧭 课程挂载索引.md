# 🧭 课程挂载索引

> 目标是“尽量复用现有主路径，只有真正缺口才新增路径”。重复课程不重复建目录，只补索引入口。

---

## 📌 本轮处理原则

| 原则 | 说明 |
|:---|:---|
| **能并入就并入** | 已有成熟学习路径的课程，直接挂到对应路径，不重复新增 |
| **同类课程合并** | 名称接近、核心内容重叠的课程，用同一条路径承接 |
| **明显缺口单独补** | 数据挖掘 / 检索 / 大数据 / 可视化作为独立新路径补齐 |
| **入口先行** | 先把索引打通，后续再按需要逐步展开原子笔记 |
