# 🗺️ 概率论与数理统计学习路径

> 这条 V2 试点路径的目标，是把 `概率语言 -> 抽样理论 -> 统计推断 -> 渐近与应用` 变成一条可执行、可复盘的学习主线。

```dataviewjs
const stages = [
  { name: "🎲 第一阶段", path: '"🧠 学习系统/22-概率论与数理统计/01-概率基础"' },
  { name: "📊 第二阶段", path: '"🧠 学习系统/22-概率论与数理统计/02-抽样与分布理论"' },
  { name: "📈 第三阶段", path: '"🧠 学习系统/22-概率论与数理统计/03-统计推断"' },
  { name: "🚀 第四阶段", path: '"🧠 学习系统/22-概率论与数理统计/04-渐近理论与专题应用"' }
];

let total = 0;
let completed = 0;
const rows = stages.map(stage => {
  const notes = dv.pages(stage.path).where(p => ["study", "project"].includes(p.type));
  const done = notes.where(p => p.status === "completed").length;
  total += notes.length;
  completed += done;
  return [stage.name, notes.length, done, notes.length ? `${Math.round(done * 100 / notes.length)}%` : "—"];
});

dv.table(["阶段", "原子卡数", "已完成", "完成率"], rows);
dv.paragraph(`**总体进度：${total ? Math.round(completed * 100 / total) : 0}%** | 已完成 ${completed} / ${total} 张任务卡`);
```

## 🪜 推荐推进顺序

1. 第一阶段先把事件、分布、数字特征和极限定理吃透。
2. 第二阶段把“样本为什么能推总体”这件事讲清楚。
3. 第三阶段再把区间估计、检验、似然和线性模型入口连成一套。
4. 最后进入第四阶段，把经典推断推广到大样本、稳健性与应用表达。
