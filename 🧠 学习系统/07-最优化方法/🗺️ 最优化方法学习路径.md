# 🗺️ 最优化方法学习路径

> 这条 V2 试点路径的目标，不是把算法名词铺满，而是把 `数学底座 -> 凸性与最优性 -> 经典数值算法 -> 随机优化 -> 应用专题` 组织成一条真正可执行的训练路线。

```dataviewjs
const stages = [
  { name: "🔧 第一阶段", path: '"🧠 学习系统/07-最优化方法/01-数学基础"' },
  { name: "📐 第二阶段", path: '"🧠 学习系统/07-最优化方法/02-凸分析与最优性条件"' },
  { name: "⚙️ 第三阶段", path: '"🧠 学习系统/07-最优化方法/03-经典数值优化算法"' },
  { name: "🚀 第四阶段", path: '"🧠 学习系统/07-最优化方法/04-现代一阶与随机优化"' },
  { name: "🎯 第五阶段", path: '"🧠 学习系统/07-最优化方法/05-专题应用"' }
];

let total = 0;
let completed = 0;
const rows = stages.map(stage => {
  const notes = dv.pages(stage.path).where(p => ["study", "project"].includes(p.type));
  const done = notes.where(p => p.status === "completed").length;
  total += notes.length;
  completed += done;
  return [stage.name, notes.length, done, notes.length ? `${Math.round(done * 100 / notes.length)}%` : "—"];
});

dv.table(["阶段", "原子卡数", "已完成", "完成率"], rows);
dv.paragraph(`**总体进度：${total ? Math.round(completed * 100 / total) : 0}%** | 已完成 ${completed} / ${total} 张任务卡`);
```

## 🪜 推荐推进顺序

1. 先把第一阶段的数学底座补齐，再进入第二阶段的凸性主线。
2. 第三阶段优先吃透线搜索、牛顿/拟牛顿、信赖域三条经典数值路线。
3. 第四阶段转向随机优化与近端方法，理解现代训练为何不再只靠牛顿法。
4. 第五阶段再把深度学习、组合优化、RL、联邦学习等专题接进来。
