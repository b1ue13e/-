# 🧭 时间序列分析专题支线导航

> 主线回答“时间序列该按什么教材顺序学”，支线回答“你更想优先解决哪一类时序问题”。

## 🗺️ 支线总览

| 支线 | 适合目标 | 主抓板块 | 难度 | 典型产出 |
|:---|:---|:---|:---|:---|
| 📈 经典统计预测 | 做稳定、可解释的单变量预测 baseline | B + C + D | 中 | ARIMA/SARIMA 预测报告、模型对比表 |
| 🌦️ 趋势季节与结构分解 | 做趋势、季节性、节假日和结构变化分析 | A + D | 中 | 分解 notebook、季节性诊断清单 |
| 🔗 非平稳与协整关系 | 做单位根、协整、误差修正和多序列长期关系研究 | B + D + F | 高 | 协整检验报告、ECM 研究笔记 |
| 💹 波动率与金融时序 | 做收益率、异方差、波动率和风险动态建模 | C + D | 高 | ARCH/GARCH 实验、波动率摘要 |
| 🤖 时序特征工程与机器学习 | 做工业预测、特征工程和滚动验证 pipeline | E + F | 中高 | 时序 ML baseline、walk-forward 框架 |
| 🧠 深度学习时序预测 | 做 RNN、LSTM、Transformer、N-BEATS 这类神经时序模型 | E + F | 高 | 深度时序对比实验、模型卡 |
| 🎲 高斯过程与概率时序 | 做小样本、非参数、不确定性表达和概率预测 | A + E | 高 | 概率预测笔记、GP/DGP 对照实验 |

## 📈 支线1：经典统计预测

| 维度 | 说明 |
|:---|:---|
| 学习目标 | 用最经典、最稳的统计路线完成从“识别 -> 估计 -> 诊断 -> 预测”的单变量时序闭环。 |
| 适合你如果 | 你想先建立一个靠谱 baseline，做销量、流量、经济指标、宏观指标预测，不想一上来就用深度学习。 |
| 建议前置 | [[01-基础认知层/🎯 板块A：基础认知层\|板块A]] 与 [[02-统计基础层/📊 板块B：统计基础层\|板块B]]。 |
| 核心模块 | 1. 统计语言：[[02-统计基础层/平稳时间序列\|平稳时间序列]]、[[02-统计基础层/自相关函数ACF\|ACF]]、[[02-统计基础层/白噪声\|白噪声]]。<br>2. 模型主干：[[03-经典模型层/AR模型\|AR模型]]、[[03-经典模型层/MA模型\|MA模型]]、[[03-经典模型层/ARMA模型\|ARMA模型]]、[[03-经典模型层/ARIMA模型\|ARIMA模型]]、[[03-经典模型层/SARIMA模型\|SARIMA模型]]。<br>3. 建模流程：[[03-经典模型层/模型识别\|模型识别]]、[[03-经典模型层/参数估计\|参数估计]]、[[03-经典模型层/AIC与BIC\|AIC与BIC]]、[[03-经典模型层/预测与评估\|预测与评估]]。<br>4. 衔接诊断：残差检查、差分选择、季节项解释。 |
| 板块映射 | `B` 管统计基础，`C` 管模型主体，`D` 管诊断与修正。 |
| 建议产出 | 一份 ARIMA/SARIMA baseline 报告、一张模型选择清单、一套可复用的预测评估模板。 |
| 强关联路径 | [[🧠 学习系统/30-应用回归分析/🗺️ 应用回归分析学习路径\|应用回归分析]]、[[🎓 时间序列建模实战路线\|时间序列建模实战路线]]。 |

## 🔗 支线3：非平稳与协整关系

| 维度 | 说明 |
|:---|:---|
| 学习目标 | 处理“序列看起来相关，但其实是假关系”的问题，建立长期均衡与短期偏离的分析框架。 |
| 适合你如果 | 你关注宏观、金融、经济计量或多变量长期关系，不满足于只做单条序列预测。 |
| 建议前置 | [[02-统计基础层/平稳性检验\|平稳性检验]]、[[02-统计基础层/单位根检验\|单位根检验]]、[[03-经典模型层/差分运算\|差分运算]]。 |
| 核心模块 | 1. 非平稳识别：[[02-统计基础层/平稳性的统计性质\|平稳性的统计性质]]、[[02-统计基础层/单位根检验\|单位根检验]]。<br>2. 伪关系排查：[[04-建模与诊断层/虚假回归\|虚假回归]]、残差行为检查。<br>3. 长期关系：[[04-建模与诊断层/协整\|协整]]、长期均衡解释。<br>4. 动态修正：[[04-建模与诊断层/误差修正模型\|误差修正模型]]、[[04-建模与诊断层/残差自回归模型\|残差自回归模型]]。 |
| 板块映射 | `B` 负责检验，`D` 负责建模，`F` 适合做真实多序列项目沉淀。 |
| 建议产出 | 一份单位根与协整检验报告、一套 ECM 模型说明、一页“虚假回归排查流程”。 |
| 强关联路径 | [[🧠 学习系统/03-量化金融/🗺️ 量化金融学习路径\|量化金融]]、[[🧠 学习系统/24-随机过程/🗺️ 随机过程学习路径\|随机过程]]、[[🧠 学习系统/22-概率论与数理统计/🗺️ 概率论与数理统计学习路径\|概率统计]]。 |

## 💹 支线4：波动率与金融时序

| 维度 | 说明 |

| 建议前置 | [[03-经典模型层/ARIMA模型\|ARIMA模型]]、[[04-建模与诊断层/方差齐性变换\|方差齐性变换]]，并建议并读 [[🧠 学习系统/03-量化金融/🗺️ 量化金融学习路径\|量化金融]]。 |

| 建议产出 | 一条时序特征工程模板、一套滚动验证脚本、一份统计模型 vs ML 模型对比报告。 |
| 强关联路径 | [[🧠 学习系统/17-机器学习/🗺️ 机器学习学习路径\|机器学习]]、[[🧠 学习系统/36-数据挖掘与信息检索/🗺️ 数据挖掘与信息检索学习路径\|数据挖掘与信息检索]]。 |

| 核心模块 | 1. 序列网络：[[05-机器学习与深度学习进阶/RNN时序建模\|RNN时序建模]]、[[05-机器学习与深度学习进阶/LSTM与GRU\|LSTM与GRU]]。<br>2. 长依赖建模：[[05-机器学习与深度学习进阶/Transformer时序\|Transformer时序]]。<br>3. 现代预测架构：[[05-机器学习与深度学习进阶/N-BEATS与N-HiTS\|N-BEATS 与 N-HiTS]]。<br>4. 评估策略：与统计模型、树模型做多基线比较，避免只看单次分数。 |

| 建议产出 | 一份 RNN/LSTM/Transformer/N-BEATS 对比实验、一张模型适用场景表、一套神经时序训练记录模板。 |

## 🎲 支线7：高斯过程与概率时序

| 维度 | 说明 |
|:---|:---|
| 学习目标 | 引入不确定性表达、非参数函数建模和小样本时序视角，而不是只做点预测。 |
| 适合你如果 | 你更在意预测区间、样本较少、采样不规则，或者希望掌握更“概率化”的时序建模路线。 |
| 建议前置 | [[01-基础认知层/统计时序分析\|统计时序分析]]、[[01-基础认知层/三次样条插值\|三次样条插值]]，最好已经具备经典时序 baseline。 |
| 核心模块 | 1. 非参数视角：函数先验、核函数、平滑与不确定性。<br>2. 模型入口：[[01-基础认知层/高斯过程回归\|高斯过程回归]]、[[01-基础认知层/Deep Gaussian Processes (DGP)\|Deep Gaussian Processes]]。<br>3. 数据条件：不规则采样、小样本、高噪声环境。<br>4. 比较框架：与 ARIMA、XGBoost、LSTM 的适用边界比较。 |
| 板块映射 | `A` 提供概念入口，`E` 适合把它放进现代时序方法对照组里。 |
| 建议产出 | 一份概率预测说明、一页 GP/DGP 对照卡、一份小样本时序实验笔记。 |
| 强关联路径 | [[🧠 学习系统/33-贝叶斯统计/🗺️ 贝叶斯统计学习路径\|贝叶斯统计]]、[[🧠 学习系统/32-非参数统计/🗺️ 非参数统计学习路径\|非参数统计]]、[[🧠 学习系统/17-机器学习/🗺️ 机器学习学习路径\|机器学习]]。 |
