## 🧭 当前处理规则

| 规则 | 说明 |
|:---|:---|
| 只挂载本地已有或明确可用的教材 | 先把你已经放进库里的 PDF/笔记接入路径 |
| 先做“问题 -> 教材 -> 阶段”闭环 | 不只列书单，而是说明这本书解决什么问题、该先看哪里 |
| 缺书时先记缺口，不直接乱抓 | 缺少教材时先放进待补清单，再决定补本地资源还是仅做索引 |
| 每条路径按自己的阶段结构挂载 | 不强行把所有学科压成同一种四阶段模板 |
| 缺口统一进下载清单 | 待下载教材统一汇总到 [[📥 PDF教材待下载清单\|PDF教材待下载清单]] |
