# 🗺️ 高等代数学习路径

> 这条 V2 试点路径不再只告诉你“有哪些知识点”，而是把 `教材章节 -> 模块问题 -> 原子任务 -> 完成标准` 串成一条可执行学习主线。

```dataviewjs
const stages = [
  { name: "📐 第一阶段", path: '"🧠 学习系统/09-高等代数/01-基础入门"' },
  { name: "🎨 第二阶段", path: '"🧠 学习系统/09-高等代数/02-理论深化"' },
  { name: "🌟 第三阶段", path: '"🧠 学习系统/09-高等代数/03-进阶提升"' },
  { name: "🚀 第四阶段", path: '"🧠 学习系统/09-高等代数/04-精通拓展"' }
];

let total = 0;
let completed = 0;
const rows = stages.map(stage => {
  const notes = dv.pages(stage.path).where(p => ["study", "project"].includes(p.type));
  const done = notes.where(p => p.status === "completed").length;
  total += notes.length;
  completed += done;
  return [stage.name, notes.length, done, notes.length ? `${Math.round(done * 100 / notes.length)}%` : "—"];
});

dv.table(["阶段", "原子卡数", "已完成", "完成率"], rows);
dv.paragraph(`**总体进度：${total ? Math.round(completed * 100 / total) : 0}%** | 已完成 ${completed} / ${total} 张任务卡`);
```

## 🪜 推荐推进顺序

1. 先完成 `第一阶段`，把行列式、矩阵、方程组、秩打成一个整体。
2. 再完成 `第二阶段`，优先吃透“空间 - 映射 - 特征值”这条主线。
3. 第三阶段集中解决正交、Jordan、二次型三个结构拐点。
4. 最后再进入第四阶段，把高代接到数值计算、PCA、谱方法和现代应用。
