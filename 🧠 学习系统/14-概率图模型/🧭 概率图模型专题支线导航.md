# 🧭 概率图模型专题支线导航

> 主线回答“概率图模型按什么顺序系统学”，支线回答“你更想优先吃透哪一类图模型问题”。

## 🗺️ 支线总览

| 支线 | 适合目标 | 主抓阶段 | 难度 | 典型产出 |
|:---|:---|:---|:---|:---|
| 🎨 图表示与条件独立 | 打牢 BN / MRF / 因子分解 / 条件独立语言 | 第一阶段 | 中高 | 图结构速查表、独立性判读卡 |
| 🔄 精确推断与消息传递 | 系统掌握 VE、联结树、和积/最大积、BP | 第二阶段 | 高 | 推断算法对比表、VE/JT 实现 |
| 🎲 采样与变分推断 | 掌握 MCMC、VI、ELBO 和近似推断路线 | 第二阶段 | 很高 | 采样诊断笔记、VI 实验模板 |
| 📚 参数学习与结构学习 | 做参数估计、EM、结构搜索和 Chow-Liu | 第三阶段 | 高 | BN 学习流水线、EM 对照实验 |
| 🌐 因果推断与贝叶斯网络 | 用 BN 做因果图、干预、反事实和动态 BN | 第四阶段 | 很高 | 因果估计模板、医疗BN案例 |
| 🔗 序列图模型与结构化预测 | 学 HMM、CRF、Viterbi、NER/POS 等序列建模 | 第五阶段 | 高 | HMM/CRF 对照笔记、序列标注实验 |
| 🚀 生成模型、主题模型与概率编程 | 把 PGM 过渡到 LDA、VAE、扩散和概率编程 | 第六阶段 | 很高 | 生成模型路线图、概率编程工具卡 |

## 🎨 支线1：图表示与条件独立

| 维度 | 说明 |
|:---|:---|
| 学习目标 | 把概率图模型最核心的“结构表达能力”学扎实，建立有向图、无向图、因子分解和条件独立的统一视角。 |
| 适合你如果 | 你现在一看到 BN、MRF、因子图就混淆，或者想先把 PGM 的语言体系学清楚。 |
| 建议前置 | 概率基础、条件概率、联合分布基础即可。 |
| 核心模块 | 1. 基本表达：[[01-基础与表示/联合分布\|联合分布]]、[[01-基础与表示/条件概率\|条件概率]]、[[01-基础与表示/因子分解\|因子分解]]。<br>2. BN 主线：[[01-基础与表示/BN结构\|BN结构]]、[[01-基础与表示/BN分解\|BN分解]]、[[01-基础与表示/D分离\|D分离]]。<br>3. MRF 主线：[[01-基础与表示/MRF结构\|MRF结构]]、[[01-基础与表示/局部马尔可夫MRF\|局部马尔可夫MRF]]、[[01-基础与表示/Gibbs分布\|Gibbs分布]]、[[01-基础与表示/HC定理\|HC定理]]。<br>4. 结构判读：[[01-基础与表示/链式结构\|链式结构]]、[[01-基础与表示/分叉结构\|分叉结构]]、[[01-基础与表示/汇连结构\|汇连结构]]、[[01-基础与表示/全局马尔可夫\|全局马尔可夫]]。 |
| 阶段映射 | 完全锚定 `第一阶段`，是所有推断、学习、因果和序列模型的共同地基。 |
| 建议产出 | 一份 BN / MRF / 因子图对照表、一张条件独立判读速查卡、一页“什么时候选有向图，什么时候选无向图”的总结。 |
| 强关联路径 | [[🧠 学习系统/17-机器学习/🗺️ 机器学习学习路径\|机器学习]]、[[🧠 学习系统/01-社交网络挖掘/🗺️ 社交网络挖掘学习路径\|社交网络挖掘]]。 |

## 🔄 支线2：精确推断与消息传递

| 维度 | 说明 |
|:---|:---|
| 学习目标 | 学会在图结构上高效做边际概率、MAP 和后验计算，掌握“为什么图能把复杂推断拆开”。 |
| 适合你如果 | 你最关心概率查询如何真正算出来，而不是先学模型训练。 |
| 建议前置 | 支线1 的图表示与条件独立必须先通。 |
| 核心模块 | 1. 精确推断起点：[[02-推断算法/因子运算\|因子运算]]、[[02-推断算法/变量消去\|变量消去]]、[[02-推断算法/消去顺序\|消去顺序]]。<br>2. 消息传递：[[02-推断算法/消息传递\|消息传递]]、[[02-推断算法/和积算法\|和积算法]]、[[02-推断算法/最大积\|最大积]]。<br>3. 联结树：[[02-推断算法/联结树构建\|联结树构建]]、[[02-推断算法/联结树BP\|联结树BP]]、[[02-推断算法/三角化\|三角化]]。<br>4. 项目巩固：[[02-推断算法/项目-VE实现\|VE实现]]、[[02-推断算法/项目-推断对比\|推断对比]]。 |
| 阶段映射 | 主体在 `第二阶段`，是 PGM 中“会表示”到“会计算”的第一步。 |
| 建议产出 | 一份 VE / BP / JT 对照表、一套精确推断实现、一页“图结构如何影响推断复杂度”的总结。 |
| 强关联路径 | [[🧠 学习系统/10-数据结构与算法/🗺️ 数据结构与算法学习路径\|数据结构与算法]]、[[🧠 学习系统/22-概率论与数理统计/🗺️ 概率论与数理统计学习路径\|概率统计]]。 |
