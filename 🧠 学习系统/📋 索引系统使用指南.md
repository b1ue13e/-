# 📋 知识索引系统使用指南

> 本系统采用**四层索引架构**，从宏观到微观，从人工到自动，构建完整的知识网络。

```
┌─────────────────────────────────────────────────────────────────┐
│                    🌐 知识索引系统架构                           │
├─────────────────────────────────────────────────────────────────┤
│                                                                  │
│  第一层：宏观架构层 (Macro)                                       │
│  ├── 🌐 知识网络中枢 - 所有路径的统一入口和交叉索引              │
│  └── 功能：路径规划、推荐组合、知识网络可视化                    │
│                                                                  │
│  第二层：学习路径层 (Meso)                                        │
│  ├── 🔀 知识连接索引 - 各学习路径底部的跨路径关联                │
│  └── 功能：路径间微观关联、概念映射                              │
│                                                                  │
│  第三层：概念聚合层 (Concept)                                     │
│  ├── 🔍 概念自动索引 - 按concept字段自动聚合                     │
│  └── 功能：跨路径概念整合、相似概念对比                          │
│                                                                  │
│  第四层：依赖分析层 (Dependency)                                  │
│  ├── 🕸️ 知识依赖图谱 - 基于prerequisites的依赖网络              │
│  └── 功能：前置知识检查、学习顺序推荐、缺口分析                  │
│                                                                  │
│  辅助索引：                                                        │
│  └── 🏷️ 标签云索引 - 基于tags的多维度分类筛选                    │
│                                                                  │
└─────────────────────────────────────────────────────────────────┘
```

## 🌐 第一层：知识网络中枢

### 核心功能

1. **学习路径全图** - 可视化17条路径的层级关系
2. **跨路径知识连接矩阵** - 数学工具→AI应用的映射
3. **概念云** - 按出现频率展示核心概念
4. **学习路径推荐引擎** - 按目标推荐路径组合
5. **前置知识缺口分析** - 智能检测缺失的基础

### 使用示例

> **场景**：你想成为机器学习工程师，但不知道从何开始

1. 打开 [[🌐 知识网络中枢]]
2. 查看"🎓 学习路径推荐引擎"表格
3. 找到"成为机器学习工程师"行
4. 查看推荐的学习顺序：多元统计 → 最优化 → 机器学习 → CV/NLP/RL
5. 点击对应路径开始学习

## 🔍 第三层：概念自动索引

### 核心功能

1. **概念热度云** - 50个高频概念的可视化云
2. **概念分类导航** - 数学基础/机器学习/深度学习分类
3. **概念详细索引** - 每个概念下的所有笔记列表
4. **概念对比工具** - 相似概念的辨析表格
5. **概念学习路径** - 推荐的学习顺序

> **场景**：你想系统学习"梯度下降"

1. 打开 [[🔍 概念自动索引]]
2. 在概念云中找到"梯度下降"（字号越大表示越重要）
3. 点击跳转到该概念详细索引
4. 查看涉及的所有笔记（优化方法、机器学习、深度学习等路径）
5. 查看相关概念（SGD、Adam、学习率调度）
6. 在"概念对比工具"中对比梯度下降vs牛顿法

## 📝 如何让索引为你工作

### 1. 养成标记 frontmatter 的习惯

```yaml
---
# 基本信息
topic: "Adam优化器"                    # 知识点名称（用于索引）
status: completed                      # 学习状态（用于进度统计）
difficulty: intermediate               # 难度（用于推荐）

# 索引字段（核心！）
concept: ["梯度下降", "优化算法", "自适应学习率"]  # 概念标签（用于概念索引）
prerequisites: ["梯度下降基础", "指数移动平均"]   # 前置知识（用于依赖图谱）
related: ["SGD", "RMSprop", "学习率调度"]        # 相关概念（用于知识网络）

# 分类标签
tags: [学习, 优化算法, 机器学习, intermediate]
---
```

### 2. 定期查看索引页面

| 频率 | 查看页面 | 目的 |
|:---|:---|:---|
| 每周 | 🕸️ 知识依赖图谱 | 检查知识缺口，规划下周学习 |
| 每月 | 🌐 知识网络中枢 | 审视整体进度，调整学习路线 |
| 学习新概念前 | 🔍 概念自动索引 | 了解概念全貌和相关知识 |
| 学习后 | 🔍 概念自动索引 | 更新笔记的 concept/prerequisites |

### 3. 利用索引进行复习

```dataviewjs
// 示例：查找已学过但很久没复习的概念
const reviewCandidates = dv.pages('"🧠 学习系统"')
    .where(p => p.status === "completed")
    .where(p => p.file.mtime < dv.date('now').minus({days: 30}))
    .sort(p => p.file.mtime)
    .limit(10);

// 这些就是需要复习的内容
```

## 🔧 故障排除

### 笔记没有出现在索引中？

**检查清单**：
- [ ] frontmatter 是否以 `---` 开头和结尾
- [ ] `topic` 字段是否填写
- [ ] `concept` 是否为数组格式 `["概念1", "概念2"]`
- [ ] 文件是否保存在 `🧠 学习系统` 目录下
- [ ] 是否保存文件（Obsidian 需要保存后才能索引）

### 概念索引为空？

**可能原因**：
1. 笔记中没有 `concept` 字段
2. `concept` 字段格式错误（不是数组）
3. Dataview 插件未启用 JavaScript 查询

**解决方法**：
1. 检查并添加 `concept: ["概念1", "概念2"]`
2. 在 Dataview 设置中启用 JavaScript

### 依赖图谱不显示缺口？

**可能原因**：
1. 笔记中没有 `prerequisites` 字段
2. 前置知识笔记的 `topic` 与 `prerequisites` 中的名称不匹配

**解决方法**：
1. 确保前置知识笔记的 `topic` 与依赖笔记的 `prerequisites` 完全一致
2. 使用标准概念名称（如都用"梯度下降"而非混用"GD"和"梯度下降"）

## 🎨 索引系统进阶技巧

### 1. 自定义概念词汇表

为了保持索引一致性，建议建立个人概念词汇表：

```markdown
## 标准概念命名规范

- ✅ "梯度下降"（不要用 GD、gradient descent）
- ✅ "主成分分析"（不要用 PCA、Principal Component Analysis）
- ✅ "支持向量机"（不要用 SVM）
- ✅ "神经网络"（不要用 NN、Neural Network）
```

### 2. 利用双向链接增强索引

在笔记正文中使用 `概念名` 创建双向链接：

```markdown
Adam优化器结合了 [[动量法]] 和 [[RMSprop]] 的优点...
```

这样可以在图谱视图中看到概念间的连接。

### 3. 定期整理索引

每月进行一次索引整理：

1. 检查 [[🔍 概念自动索引]] 中的低频概念
2. 合并相似概念（如"SGD"和"随机梯度下降"）
3. 更新 [[🌐 知识网络中枢]] 中的连接矩阵
