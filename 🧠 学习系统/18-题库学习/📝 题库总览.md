# 📝 题库学习系统

> 从“模板堆放区”升级为“知识点挂题 + 复习驱动”的结构化题库。

---

## 📊 题库统计

```dataviewjs
const excludedStudyDirs = [
    "🧠 学习系统/18-题库学习/",
    "🧠 学习系统/19-习惯追踪/",
    "🧠 学习系统/20-PARA知识管理/",
    "🧠 学习系统/21-周期笔记/"
];
const isTemplate = p =>
    p.file.name.includes("模板") ||
    (Array.isArray(p.tags) && p.tags.includes("模板"));
const toArray = value => Array.isArray(value) ? value : (value ? [value] : []);
const toPath = value => {
    if (!value) return "";
    if (typeof value === "string") return value.replace(/\.md$/, "");
    if (value.path) return value.path.replace(/\.md$/, "");
    return String(value).replace(/\.md$/, "");
};

const questionPages = dv.pages('"🧠 学习系统/18-题库学习"')
    .where(p => p.type === "question" && !isTemplate(p))
    .array();
const studyPages = dv.pages('"🧠 学习系统"')
    .where(p =>
        p.type === "study" &&
        !excludedStudyDirs.some(dir => p.file.path.startsWith(dir))
    )
    .array();
const linkedNotePaths = new Set(
    questionPages.flatMap(p => toArray(p.linked_notes).map(toPath).filter(Boolean))
);
const notesWithPractice = studyPages.filter(p =>
    linkedNotePaths.has(p.file.path.replace(/\.md$/, ""))
).length;
const countByType = type => questionPages.filter(p => p.question_type === type).length;
const weakQuestions = questionPages.filter(p => Number(p.mistake_count || 0) > 0).length;

dv.table(
    ["指标", "数量"],
    [
        ["题目总数", questionPages.length],
        ["单选题", countByType("single_choice")],
        ["多选题", countByType("multiple_choice")],
        ["判断题", countByType("true_false")],
        ["主观题", countByType("essay")],
        ["已挂题学习笔记", notesWithPractice],
        ["仍有错题记录的题目", weakQuestions]
    ]
);
```

## 🧭 使用规则

1. 每道题都至少绑定一个 `linked_notes:: [[知识笔记]]`。
2. 题目状态统一使用 `unseen / practicing / mastered`。
3. 错题优先更新 `mistake_count` 和 `last_practiced`，不要只改正文。
4. 题目创建完成后，把题链回对应学习笔记的“练习闭环”区域。

## 📚 最近训练题

```dataview
TABLE question as "题目", question_type as "题型", status as "状态", mistake_count as "错题次数", linked_notes as "关联笔记"
FROM "🧠 学习系统/18-题库学习"
WHERE type = "question" AND !contains(file.name, "模板") AND !contains(tags, "模板")
SORT last_practiced DESC, created DESC
```

## 🧩 待补练习的知识点

```dataviewjs
const excludedStudyDirs = [
    "🧠 学习系统/18-题库学习/",
    "🧠 学习系统/19-习惯追踪/",
    "🧠 学习系统/20-PARA知识管理/",
    "🧠 学习系统/21-周期笔记/"
];
const isTemplate = p =>
    p.file.name.includes("模板") ||
    (Array.isArray(p.tags) && p.tags.includes("模板"));
const toArray = value => Array.isArray(value) ? value : (value ? [value] : []);
const toPath = value => {
    if (!value) return "";
    if (typeof value === "string") return value.replace(/\.md$/, "");
    if (value.path) return value.path.replace(/\.md$/, "");
    return String(value).replace(/\.md$/, "");
};

const questionPages = dv.pages('"🧠 学习系统/18-题库学习"')
    .where(p => p.type === "question" && !isTemplate(p))
    .array();
const linkedNotePaths = new Set(
    questionPages.flatMap(p => toArray(p.linked_notes).map(toPath).filter(Boolean))
);
const notes = dv.pages('"🧠 学习系统"')
    .where(p =>
        p.type === "study" &&
        !excludedStudyDirs.some(dir => p.file.path.startsWith(dir))
    )
    .array();

const noPractice = notes
    .filter(p => Array.isArray(p.concept) && p.concept.length > 0)
    .filter(p => !linkedNotePaths.has(p.file.path.replace(/\.md$/, "")))
    .slice(0, 15);

if (noPractice.length === 0) {
    dv.paragraph("✅ 当前带概念标签的学习笔记都已经至少挂上一道题。");
} else {
    dv.table(
        ["学习笔记", "概念", "建议补题方向"],
        noPractice.map(p => [
            p.file.link,
            (p.concept || []).join(" / "),
            "先补 1 道概念辨析题 + 1 道应用题"
        ])
    );
}
```
