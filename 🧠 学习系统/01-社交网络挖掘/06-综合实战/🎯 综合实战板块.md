## 🛠️ 项目模板

### 项目结构

```
project-name/
├── data/               # 数据文件
├── notebooks/          # Jupyter notebooks
├── src/               # 源代码
│   ├── models/        # 模型定义
│   ├── utils/         # 工具函数
│   └── train.py       # 训练脚本
├── checkpoints/       # 模型检查点
├── results/           # 实验结果
├── README.md          # 项目说明
└── requirements.txt   # 依赖
```
