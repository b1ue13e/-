# 社交网络挖掘导论

> 社交网络挖掘是研究社交关系数据中的模式、结构和动态规律的交叉学科。

## 什么是社交网络挖掘

社交网络挖掘（Social Network Mining）是从社交网络数据中提取有价值信息和知识的过程。

### 核心研究对象

- **节点（Node）**：社交网络中的个体（人、组织、页面）
- **边（Edge）**：个体之间的关系（好友、关注、互动）
- **网络结构**：节点和边形成的拓扑结构

### 主要任务类型

| 任务类型 | 描述 | 示例 |
|:---|:---|:---|
| **社区发现** | 识别紧密连接的群体 | 发现朋友圈、兴趣社群 |
| **链接预测** | 预测可能形成的连接 | 好友推荐、关注推荐 |
| **影响力分析** | 识别关键节点和信息传播 | 找到 KOL、舆情监控 |
| **信息传播** | 建模信息在网络中的流动 | 病毒式营销、谣言控制 |

## 与相关学科的关系

```
社交网络挖掘
    ├── 图论（数学基础）
    ├── 机器学习（算法工具）
    ├── 社会学（理论指导）
    └── 统计学（分析方法）
```

## 应用场景

- **社交媒体分析**：用户行为理解、内容推荐
- **电商网络**：商品推荐、用户分群
- **金融风控**：欺诈检测、信用评估
- **生物信息**：蛋白质相互作用网络、基因调控网络
