## 🪜 建议阅读顺序

1. 先用 `Speech and Language Processing` 走完第一、二阶段，把语言学、传统方法和评测语言补齐。
2. 第二步切到 `Neural Network Methods for NLP` 和 `Natural Language Processing with PyTorch`，把词向量、RNN、Seq2Seq 和注意力吃透。
3. 第三步回到 `Speech and Language Processing` 的现代章节，再结合工程书推进 Transformer、预训练和 LLM。
4. 最后用 `检索力` 和 PyTorch 实践把 RAG、问答、对话和专业任务做成完整应用链。
