# 算法学习诊断对话模板

> 系统性诊断当前算法水平，制定针对性学习计划。

## 诊断维度

### 1. 基础能力评估

| 问题 | 自评 | 备注 |
|:---|:---:|:---|
| 能独立完成 Two Sum 吗？ | ⬜ | 基础数组操作 |
| 理解时间/空间复杂度分析吗？ | ⬜ | Big-O 表示法 |
| 能手写二分查找吗？ | ⬜ | 边界处理 |
| 理解递归的执行过程吗？ | ⬜ | 调用栈、基例 |

### 2. 数据结构掌握

| 数据结构 | 理解程度 | 能独立实现 |
|:---|:---:|:---:|
| 数组/链表 | ⬜ 初识 / ⬜ 熟悉 / ⬜ 精通 | ⬜ |
| 栈/队列 | ⬜ 初识 / ⬜ 熟悉 / ⬜ 精通 | ⬜ |
| 二叉树 | ⬜ 初识 / ⬜ 熟悉 / ⬜ 精通 | ⬜ |
| 哈希表 | ⬜ 初识 / ⬜ 熟悉 / ⬜ 精通 | ⬜ |
| 图 | ⬜ 初识 / ⬜ 熟悉 / ⬜ 精通 | ⬜ |

### 3. 算法思想

- [ ] 分治（归并排序、快速排序）
- [ ] 贪心（区间调度、霍夫曼编码）
- [ ] 动态规划（背包、最长子序列）
- [ ] 回溯（全排列、N皇后）
- [ ] 双指针/滑动窗口

## 诊断流程

```
1. 自评 → 2. 薄弱点识别 → 3. 制定计划 → 4. 执行 → 5. 复评
```
