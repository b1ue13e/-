## 📐 数学维度

> 控制理论的数学基础

### 核心数学工具

| 数学领域 | 关键概念 | 应用场景 |
|:--------|:--------|:--------|
| **微分方程** | ODE、PDE、状态空间表示 | 系统动态建模 |
| **拉普拉斯变换** | 传递函数、极点零点、频域分析 | 经典控制设计 |
| **线性代数** | 矩阵运算、特征值、子空间 | 状态空间分析 |
| **优化理论** | 凸优化、非线性规划、动态规划 | MPC、轨迹优化 |

## 🤖 模型维度

> 物理系统的数学抽象

### 机器人建模体系

```
┌─────────────────────────────────────────┐
│           机器人建模层次                  │
├─────────────────────────────────────────┤
│  运动学 (Kinematics)                     │
│  ├── 正运动学：关节角度 → 末端位姿         │
│  ├── 逆运动学：末端位姿 → 关节角度         │
│  └── 雅可比矩阵：速度/力映射               │
├─────────────────────────────────────────┤
│  动力学 (Dynamics)                       │
│  ├── 牛顿-欧拉法：递归动力学计算           │
│  ├── 拉格朗日法：能量法建模               │
│  └── 刚体动力学：质量、惯性、质心           │
├─────────────────────────────────────────┤
│  车辆模型                                 │
│  ├── 自行车模型：路径跟踪基础              │
│  ├── 阿克曼转向：汽车运动学                │
│  └── 动力学模型：轮胎模型、滑移            │
└─────────────────────────────────────────┘
```

### 坐标系变换
- **齐次变换矩阵**：$T = \begin{bmatrix} R & p \\ 0 & 1 \end{bmatrix}$
- **欧拉角 vs 四元数**：姿态表示与万向节锁
- **DH参数法**：标准机器人运动学描述

### 学习路径关联

## 🎛️ 控制维度

> 从理论到实践的控制系统设计

### 控制方法谱系

```
                    控制方法发展谱系
                         │
        ┌────────────────┼────────────────┐
        ▼                ▼                ▼
   ┌─────────┐     ┌─────────┐     ┌─────────┐
   │ 经典控制 │     │ 现代控制 │     │ 先进控制 │
   │(SISO)   │     │(MIMO)   │     │         │
   └────┬────┘     └────┬────┘     └────┬────┘
        │               │               │
   • PID控制        • 状态反馈       • MPC
   • 根轨迹法       • 观测器设计      • 自适应控制
   • 频域设计       • LQR/LQG       • 鲁棒控制
   • 超前滞后补偿    • 卡尔曼滤波      • 智能控制
```

### 核心控制方法

| 控制类型 | 代表算法 | 适用场景 |
|:--------|:--------|:--------|
| **经典控制** | PID、Lead-Lag、根轨迹 | 单输入单输出系统 |
| **现代控制** | LQR、极点配置、状态观测器 | 多变量系统 |
| **预测控制** | MPC、LQR-MPC、Tube-MPC | 约束优化问题 |
| **状态估计** | 卡尔曼滤波、粒子滤波、EKF | 传感器融合 |

### MPC 核心流程
```
测量/估计 ──► 状态估计 ──► 优化求解 ──► 执行控制
    ▲                              │
    └──────── 滚动时域 ◄───────────┘
```

### 学习路径关联
- `07-最优化方法` - MPC的优化基础
- `13-强化学习` - 数据驱动控制方法
- `14-概率图模型` - 状态估计与贝叶斯滤波
