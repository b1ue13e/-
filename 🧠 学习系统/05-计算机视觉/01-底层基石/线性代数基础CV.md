# 计算机视觉中的线性代数

> 图像是矩阵，视觉计算是矩阵运算。

## 图像的矩阵表示

```
灰度图像：H × W 矩阵
彩色图像：H × W × 3 张量
```

## 核心概念

### 1. 向量空间

- **像素空间**：所有可能图像构成的高维空间
- **子空间**：具有特定结构的图像集合
- **基**：表示图像的基本元素

### 2. 矩阵运算

| 运算 | CV应用 | 说明 |
|:---|:---|:---|
| **矩阵乘法** | 线性变换 | 旋转、缩放、投影 |
| **特征分解** | PCA降维 | 提取主成分 |
| **SVD** | 图像压缩 | 低秩逼近 |
| **卷积** | 特征提取 | 滤波器响应 |

### 3. 几何变换

- **刚体变换**：旋转 + 平移
- **相似变换**：增加均匀缩放
- **仿射变换**：保持平行性
- **透视变换**：通用投影

## 应用实例

- **图像对齐**：通过变换矩阵对齐图像
- **降维**：PCA 减少特征维度
- **风格迁移**：在特征空间进行操作
