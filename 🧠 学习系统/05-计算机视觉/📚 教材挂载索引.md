## 🪜 建议阅读顺序

1. 先补 `Computer Vision: Algorithms and Applications`，同时用 `Mathematics for Machine Learning` 打好第一阶段的数学和表示底座。
2. 第二步进入 `Deep Learning` + `Hands-on Machine Learning`，把第二阶段的训练、优化和实验框架补齐。
3. 第三步用 `动手学深度学习` 和视觉主教材推进分类、检测、分割等任务。
4. 最后再补 `Multiple View Geometry`，把第五阶段的 3D 视觉与工程落地做深。
