# ⚡ Week 03 概率论与数理统计速通日计划

> 适用日期：**2026-04-30 ~ 2026-05-06**
> 
> 对应周次：[[✅ 2026-2027 每周执行清单版|Week 03 所在周清单]]

---

## 这周的真实目标

这不是“七天啃完整套概率统计教材”，而是“七天建立概率统计最关键的骨架”。

本周结束时，你只需要做到：

1. 能解释概率空间、随机变量、期望、方差这组最基本语言。
2. 能理解抽样分布、点估计、区间估计、假设检验在统计推断里的位置。
3. 能说清概率统计为什么是多元统计、机器学习、时间序列和量化金融的共同底层。
4. 能把“分布 -> 抽样 -> 估计 -> 检验 -> 渐近”串成一条完整主线。

本周**不要求**：

- 深入所有分布族的严格推导
- 把每个检验量公式背到滚瓜烂熟
- 把阶段页里的所有主题都展开成原子笔记

本周要求的是：**建立语言、认清流程、形成推断直觉。**

---

## 本周主线

建议你始终围绕这条主线学：

`概率空间 -> 随机变量与分布 -> 期望方差 -> 抽样分布 -> 点估计/区间估计 -> 假设检验/似然 -> 渐近与应用`
