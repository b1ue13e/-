# 🗺️ 机器学习学习路径

> 这条 V2 试点路径以《机器学习（周志华）》为理论主干，用 `Hands-on Machine Learning` 管实验闭环，用 `Mathematics for Machine Learning` 管数学底座。

```dataviewjs
const stages = [
  { name: "🌱 第一阶段", path: '"🧠 学习系统/17-机器学习/01-基础概念"' },
  { name: "📈 第二阶段", path: '"🧠 学习系统/17-机器学习/02-线性模型"' },
  { name: "🛡️ 第三阶段", path: '"🧠 学习系统/17-机器学习/03-正则优化"' },
  { name: "🌲 第四阶段", path: '"🧠 学习系统/17-机器学习/04-树与集成"' },
  { name: "🧩 第五阶段", path: '"🧠 学习系统/17-机器学习/05-其他方法"' }
];

let total = 0;
let completed = 0;
const rows = stages.map(stage => {
  const notes = dv.pages(stage.path).where(p => ["study", "project"].includes(p.type));
  const done = notes.where(p => p.status === "completed").length;
  total += notes.length;
  completed += done;
  return [stage.name, notes.length, done, notes.length ? `${Math.round(done * 100 / notes.length)}%` : "—"];
});

dv.table(["阶段", "原子卡数", "已完成", "完成率"], rows);
dv.paragraph(`**总体进度：${total ? Math.round(completed * 100 / total) : 0}%** | 已完成 ${completed} / ${total} 张任务卡`);
```

## 🪜 推荐推进顺序

1. 先完成 `第一阶段 + 第二阶段`，建立统一的任务、评估和线性建模框架。
2. 再完成 `第四阶段`，形成树模型与集成学习主线。
3. 第三阶段回过头来补“为什么能泛化、为什么要正则化”。
4. 最后系统补齐第五阶段，再决定往概率图模型或强化学习外拓。
