# 📊 项目追踪

> 项目系统现在分成两层：**模板负责规范，追踪页负责聚合。**

---

## 🔥 焦点项目（最多 7 个）

```dataviewjs
const activeStatuses = ["active", "ongoing"];
const focusOrder = { now: 0, next: 1, later: 2 };
const toMillis = value => {
    if (!value) return Number.MAX_SAFE_INTEGER;
    if (value.ts) return value.ts;
    const parsed = new Date(value);
    return Number.isNaN(parsed.getTime()) ? Number.MAX_SAFE_INTEGER : parsed.getTime();
};

const projects = dv.pages()
    .where(p => p.type === "project" && !p.template && activeStatuses.includes(p.status))
    .array()
    .sort((a, b) => {
        const focusDelta = (focusOrder[a.focus ?? "later"] ?? 9) - (focusOrder[b.focus ?? "later"] ?? 9);
        if (focusDelta !== 0) return focusDelta;
        return toMillis(a.deadline) - toMillis(b.deadline);
    });

dv.table(
    ["项目", "类型", "领域", "进度", "截止", "焦点"],
    projects.slice(0, 7).map(p => [
        p.file.link,
        p.project_type || "未分类",
        p.domain || "-",
        p.progress || "0%",
        p.deadline || "-",
        p.focus || "later"
    ])
);
```

## 🗂️ 活跃项目分类

```dataviewjs
const activeStatuses = ["active", "ongoing"];
const counts = {};
dv.pages()
    .where(p => p.type === "project" && !p.template && activeStatuses.includes(p.status))
    .forEach(p => {
        const key = p.project_type || "未分类";
        counts[key] = (counts[key] || 0) + 1;
    });

dv.table(
    ["project_type", "数量"],
    Object.entries(counts)
        .sort((a, b) => b[1] - a[1])
        .map(([name, count]) => [name, count])
);
```

## ✅ 已完成项目

```dataviewjs
dv.table(
    ["项目", "类型", "完成日期", "成果"],
    dv.pages()
        .where(p => p.type === "project" && !p.template && p.status === "completed")
        .sort(p => p.completedDate, 'desc')
        .limit(10)
        .map(p => [
            p.file.link,
            p.project_type || "未分类",
            p.completedDate || "-",
            p.outcome || "-"
        ])
);
```

## ➕ 新建项目

- 直接使用：[[📁 项目模板|📁 项目模板]]
- 推荐枚举：

| 字段 | 推荐值 |
|:--|:--|
| `status` | `active / paused / completed` |
| `project_type` | `learning / research / implementation / reading` |
| `focus` | `now / next / later` |
| `domain` | `ai / math / stats / cs / paper / book / project / interdisciplinary` |

> 兼容说明：旧项目如果还在使用 `ongoing`，当前查询仍会把它视为活跃项目；后续建议逐步改为 `active`。

## 📈 项目统计

```dataviewjs
const activeStatuses = ["active", "ongoing"];
const toDate = value => {
    if (!value) return null;
    if (value.ts) return new Date(value.ts);
    const parsed = new Date(value);
    return Number.isNaN(parsed.getTime()) ? null : parsed;
};

const now = new Date();
const projects = dv.pages().where(p => p.type === "project" && !p.template).array();
const activeProjects = projects.filter(p => activeStatuses.includes(p.status));
const focusNow = activeProjects.filter(p => (p.focus || "later") === "now").length;
const completedThisMonth = projects.filter(p => {
    if (p.status !== "completed") return false;
    const completed = toDate(p.completedDate);
    return completed &&
        completed.getFullYear() === now.getFullYear() &&
        completed.getMonth() === now.getMonth();
}).length;
const overdueProjects = activeProjects.filter(p => {
    const deadline = toDate(p.deadline);
    return deadline && deadline.getTime() < now.getTime();
}).length;

dv.table(
    ["指标", "数量"],
    [
        ["活跃项目", activeProjects.length],
        ["焦点项目（now）", focusNow],
        ["本月完成", completedThisMonth],
        ["逾期项目", overdueProjects]
    ]
);
```
