# 📘 第一册：编程不难

> **目标**：把 Python 数据科学工具链搭起来，让后面六册不再卡在“环境和代码”上。

**对应资源**：[[../../书籍笔记/鸢尾花丛书第一册-编程不难|读书笔记]] | [打开原书](<../../书籍资源/鸢尾花丛书第一册-编程不难 (姜伟生) (z-library.sk, 1lib.sk, z-lib.sk).pdf>)

---

## 🧩 本册定位

| 模块 | 关注点 | 为什么重要 |
|:---|:---|:---|
| Python 基础 | 语法、函数、面向对象 | 后面每册都会反复用到 |
| Jupyter / Anaconda | 环境、交互式学习 | 让读书、写代码、记笔记放在一起 |
| NumPy | 数组、线性代数、向量化 | 后续矩阵、统计、机器学习都靠它 |
| Pandas | 索引、规整、时间序列 | 为《数据有道》铺路 |
| Scikit-learn | 回归、降维、分类、聚类 | 给《机器学习》提前热身 |
| Streamlit | 交互 App | 把“看懂”变成“做出来” |

---

## 🎯 建议抓住的重点

- 不求语法面面俱到，先把 `NumPy + Pandas + Jupyter` 跑熟。
- 这一册里凡是出现 `Scikit-learn` 的地方，都当作第七册的预热。
- 如果你后面想做自己的数学动画或算法演示页，`Streamlit` 值得单独做一次小练习。
