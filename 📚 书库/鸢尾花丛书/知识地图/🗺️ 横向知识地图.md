## 高频交叉主题

| 主题 | 丛书落点 | 对应主路径 | 建议处理 |
|:---|:---|:---|:---|
| **Python / NumPy / Pandas / Jupyter** | 第一册、第六册、第七册 | 数据结构与算法、数据挖掘与信息检索 | 工具不重复建“学习路径”，直接以丛书为工具底座 |
| **函数 / 导数 / 积分 / 优化** | 第三册 | 数学分析、常微分方程、最优化方法 | 书里建立图像直觉，证明细节回主路径 |
| **向量 / 矩阵 / SVD / PCA** | 第四册、第五册、第六册、第七册 | 高等代数、多元统计 | 统一挂到矩阵与统计主线，不平行复制 |
| **协方差 / 高斯 / 贝叶斯** | 第五册、第六册、第七册 | 概率论与数理统计、贝叶斯统计 | 用一本到多本的方式重复巩固，知识库里交叉链接 |
| **时间数据 / 移动窗口 / 随机过程** | 第五册、第六册 | 时间序列分析、随机过程 | 本册做直观案例，后续回主路径系统学 |
| **图 / 邻接矩阵 / PageRank / 社交网络** | 第二册、第六册 | 社交网络挖掘、图神经网络 | 图论以第六册为桥，不在书库再建独立图论体系 |
| **回归 / 分类 / 聚类 / 降维** | 第一册、第五册、第六册、第七册 | 机器学习、应用回归分析、多元统计 | 丛书负责“看懂算法”，主路径负责“系统吃透” |
