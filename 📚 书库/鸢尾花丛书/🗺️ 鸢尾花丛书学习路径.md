# 🗺️ 鸢尾花丛书学习路径

> “先把数学和编程变成直觉，再把直觉变成数据分析和机器学习的手感。”

```dataviewjs
const books = dv.pages('"📚 书库/书籍笔记"').where(p => p.series === "鸢尾花丛书");
const total = books.length;
const unread = books.where(p => p.status === "unread").length;
const reading = books.where(p => p.status === "reading").length;
const finished = books.where(p => p.status === "finished").length;
const progress = total > 0 ? Math.round((finished / total) * 100) : 0;

dv.paragraph(`**总体进度: ${progress}%** | 已读完: ${finished}/${total} | 阅读中: ${reading} | 未读: ${unread}`);
```
