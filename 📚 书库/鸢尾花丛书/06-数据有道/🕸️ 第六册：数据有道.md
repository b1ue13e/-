## 🧩 本册定位

| 模块 | 关注点 | 为什么关键 |
|:---|:---|:---|
| 数据处理 | 缺失值、离群值、变换、距离度量 | 机器学习前的数据入口 |
| 时间数据 | 趋势、分解、移动窗口 | 连接时间序列分析 |
| 随机过程 | 布朗运动、几何布朗运动、高斯过程 | 连接统计与建模 |
| 图论基础 | 无向图、有向图、绘图 | 为图网络视角开门 |
| 图的分析 | 路径、连通性、中心性、社区 | 社交网络分析核心 |
| 图与矩阵 | 邻接矩阵、转移矩阵、拉普拉斯矩阵 | 串起矩阵分解和图方法 |
| 图论实践 | 树、谱聚类、PageRank、社交网络 | 直接挂到现有图相关路径 |
