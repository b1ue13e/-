## 📚 资源文件

```dataviewjs
const resources = app.vault.getFiles()
    .filter(f =>
        f.path.startsWith("📚 书库/书籍资源/") &&
        ["pdf", "epub", "mobi", "azw3", "txt"].includes((f.extension || "").toLowerCase())
    )
    .sort((a, b) => b.stat.mtime - a.stat.mtime);

if (resources.length === 0) {
    dv.paragraph("暂未发现书籍资源。");
} else {
    dv.table(
        ["文件", "格式", "修改时间"],
        resources.slice(0, 20).map(f => [
            `[${f.name}](${f.path})`,
            (f.extension || "").toUpperCase(),
            window.moment(f.stat.mtime).format("YYYY-MM-DD")
        ])
    );
}
```

## 🔗 与学习系统联动

```dataviewjs
const books = dv.pages('"📚 书库/书籍笔记"')
    .where(p => p.concept && p.concept.length > 0);

const conceptMap = {};
books.forEach(p => {
    (p.concept || []).forEach(c => {
        conceptMap[c] = (conceptMap[c] || 0) + 1;
    });
});

const hot = Object.entries(conceptMap)
    .sort((a, b) => b[1] - a[1])
    .slice(0, 10);

if (hot.length > 0) {
    dv.table(
        ["书中高频概念", "出现次数", "跳转"],
        hot.map(([name, count]) => [
            name,
            count,
            `[[🧠 学习系统/🔍 概念自动索引#${name}|查看概念索引]]`
        ])
    );
} else {
    dv.paragraph("给读书笔记补上 `concept` 后，这里会自动显示高频概念。");
}
```

## 🎯 推荐做法

1. 先在 `书籍资源` 里找到书。
2. 用 [[📘 读书笔记模板]] 建笔记。
3. 至少补 `author`、`status`、`category`、`concept`。
4. 把能迁移到学习路径里的内容，写入“与现有知识库的连接”。
